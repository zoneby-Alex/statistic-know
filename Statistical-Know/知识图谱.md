## 学习路线建议

```
第一阶段：统计基础
  概率论 → 描述统计（定量+定性）

第二阶段：核心推断
  参数估计 → 假设检验 → t检验 → 样本量估算

第三阶段：多组比较
  方差分析 → 多重比较 → 协方差分析

第四阶段：分类与等级
  卡方检验 → 非参数检验

第五阶段：回归建模
  直线回归 → 多元回归 → Logistic回归

第六阶段：设计与生存
  试验设计 → 生存分析
```
