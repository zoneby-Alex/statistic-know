# 样本量估算

## 核心概念

样本量估算是在研究设计阶段，根据预设的检验水准 $\alpha$、检验功效 $1-\beta$、效应量 $\delta$，计算出所需的最少观察对象数。

### 影响因素

| 因素 | 关系 | 说明 |
|------|------|------|
| 检验水准 $\alpha$ | 反比 | $\alpha$ 越小（如 0.01 vs 0.05），所需样本量越大 |
| 检验功效 $1-\beta$ | 正比 | 功效越高（如 0.90 vs 0.80），所需样本量越大 |
| 效应量 $\delta$ | 反比 | 效应越小，所需样本量越大 |
| 总体标准差 $\sigma$ | 正比 | 变异越大，所需样本量越大 |
| 单侧/双侧检验 | — | 双侧检验比单侧检验需要更大样本量 |

### 效应量的概念

效应量（Effect Size）是标准化的组间差异，不受样本量影响：

- **Cohen's d**：$d = \frac{|\bar{x}_1 - \bar{x}_2|}{s_{\text{pooled}}}$
- 一般标准：$d=0.2$（小）、$d=0.5$（中）、$d=0.8$（大）

---

## 均数比较的样本量公式

### 单样本均数比较

比较样本均值与已知总体均值 $\mu_0$：

$$
n = \left[ \frac{(z_{1-\alpha/2} + z_{1-\beta}) \cdot \sigma}{\delta} \right]^2
$$

其中 $\delta = |\mu - \mu_0|$ 为有临床意义的差值。

### 两独立样本均数比较

$$
n_1 = n_2 = 2 \left[ \frac{(z_{1-\alpha/2} + z_{1-\beta}) \cdot \sigma}{\delta} \right]^2
$$

---

## 率比较的样本量公式

### 两独立样本率比较

$$
n_1 = n_2 = \frac{ \left[ z_{1-\alpha/2} \sqrt{2\bar{p}(1-\bar{p})} + z_{1-\beta} \sqrt{p_1(1-p_1) + p_2(1-p_2)} \right]^2 }{(p_1 - p_2)^2}
$$

其中 $\bar{p} = (p_1 + p_2)/2$。

## Python代码示例

### 使用 statsmodels 进行样本量估算

```python
import numpy as np
from statsmodels.stats.power import (
    TTestPower,
    TTestIndPower,
    NormalIndPower,
    zt_ind_solve_power
)
from scipy import stats

# === 示例1：单样本t检验样本量 ===
# 目标：检测到效应量 d=0.5，α=0.05（双侧），功效=0.80
ttest_power = TTestPower()
n_one = ttest_power.solve_power(
    effect_size=0.5,
    alpha=0.05,
    power=0.80,
    alternative='two-sided'
)
print(f"单样本t检验所需样本量（d=0.5, power=0.80）: n = {np.ceil(n_one):.0f}")
# 输出: n = 34

# === 示例2：两独立样本t检验样本量 ===
ttest_ind = TTestIndPower()
n_two = ttest_ind.solve_power(
    effect_size=0.5,     # Cohen's d
    alpha=0.05,
    power=0.80,
    ratio=1.0,           # 两组样本量比例 n2/n1
    alternative='two-sided'
)
print(f"独立t检验每组所需样本量（d=0.5, power=0.80）: n = {np.ceil(n_two):.0f}")
# 输出: n = 64

# === 示例3：不同效应量下所需样本量的对比 ===
print("\n不同效应量下每组所需样本量（power=0.80, α=0.05）:")
print(f"{'效应量 (d)':<15} {'每组样本量':<15}")
for d in [0.2, 0.3, 0.5, 0.8, 1.0]:
    n = ttest_ind.solve_power(effect_size=d, alpha=0.05, power=0.80, ratio=1.0)
    print(f"{d:<15.1f} {np.ceil(n):<15.0f}")
# 效应量越小，所需样本量越大

# === 示例4：率比较的样本量 ===
from statsmodels.stats.proportion import proportion_effectsize

# 两组率：p1=0.60, p2=0.40
p1, p2 = 0.60, 0.40
effect = proportion_effectsize(p1, p2)
n_prop = NormalIndPower().solve_power(
    effect_size=effect,
    alpha=0.05,
    power=0.80,
    ratio=1.0,
    alternative='two-sided'
)
print(f"\n率比较（{p1:.0%} vs {p2:.0%}）每组样本量: n = {np.ceil(n_prop):.0f}")
# 输出: n ≈ 199
```

### 功效曲线绘制

```python
import matplotlib.pyplot as plt
from statsmodels.stats.power import TTestIndPower

ttest_ind = TTestIndPower()

# 固定α=0.05，绘制不同效应量下的功效曲线
fig, ax = plt.subplots(figsize=(8, 5))
ttest_ind.plot_power(
    dep_var='nobs',
    nobs=np.arange(10, 500, 10),
    effect_size=[0.2, 0.3, 0.5, 0.8],
    alpha=0.05,
    ax=ax
)
ax.set_title('独立样本t检验功效曲线（α=0.05）')
ax.set_xlabel('每组样本量')
ax.set_ylabel('统计功效 (1-β)')
ax.legend(['d=0.2', 'd=0.3', 'd=0.5', 'd=0.8'])
plt.grid(True, alpha=0.3)
plt.show()
```

### 模拟验证样本量计算结果

```python
import numpy as np
from scipy import stats

def simulate_ttest_power(d=0.5, n=64, n_sim=10000, alpha=0.05):
    """通过模拟验证t检验的实际功效"""
    count = 0
    for _ in range(n_sim):
        group1 = np.random.normal(0, 1, n)
        group2 = np.random.normal(d, 1, n)  # 均值偏移 d
        _, p = stats.ttest_ind(group1, group2)
        if p < alpha:
            count += 1
    power_est = count / n_sim
    return power_est

power_actual = simulate_ttest_power(d=0.5, n=64)
print(f"模拟实际功效（n=64, d=0.5）: {power_actual:.3f}（预期: 0.800）")
```

## 样本量估算口诀

> **样本量，看四样：**
> **检验水准α，检验功效β，**
> **效应量大小，变异程度高。**
> **α小β大效应小，变异越大样本要。**
