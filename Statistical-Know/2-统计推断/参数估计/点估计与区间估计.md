# 点估计与区间估计

## 核心概念

### 点估计 (Point Estimation)

用一个样本统计量的**单一数值**来估计总体参数。例如用样本均值 $\bar{x}$ 估计总体均值 $\mu$，用样本标准差 $s$ 估计总体标准差 $\sigma$。

**评价点估计好坏的三个标准：**

| 标准 | 含义 |
|------|------|
| 无偏性 (Unbiasedness) | $E(\hat{\theta}) = \theta$ |
| 有效性 (Efficiency) | 方差越小越有效 |
| 一致性 (Consistency) | $n \to \infty$ 时 $\hat{\theta} \to \theta$ |

### 区间估计 (Interval Estimation)

在点估计的基础上给出一个**范围**，并给出该范围包含总体参数的可信程度（置信水平）。

$$
P(\hat{\theta}_{L} < \theta < \hat{\theta}_{U}) = 1 - \alpha
$$

其中 $1 - \alpha$ 为置信水平，通常取 0.95 或 0.99。

---

## 置信区间的构造

### 总体均数的置信区间

**前提条件：** 样本来自正态分布，或样本量足够大（中心极限定理）。

#### 情况一：$\sigma$ 已知（标准正态分布）

$$
\bar{x} \pm z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}
$$

#### 情况二：$\sigma$ 未知（t分布）

$$
\bar{x} \pm t_{\alpha/2,\; n-1} \cdot \frac{s}{\sqrt{n}}
$$

### 总体率的置信区间

#### 大样本正态近似法

$$
p \pm z_{\alpha/2} \cdot \sqrt{\frac{p(1-p)}{n}}
$$

要求 $np \geq 5$ 且 $n(1-p) \geq 5$。

#### 精确法（Clopper-Pearson）

小样本时使用二项分布精确计算，详见 `scipy.stats.binom.test`。

---

## t分布与标准正态分布

### 关系对比

- 当 $n \to \infty$ 时，t分布趋近于标准正态分布
- t分布尾部更厚，相同置信水平下 t 临界值 > z 临界值，因此区间更宽
- t分布的自由度越小，与正态分布差异越大

### 临界值对比

```python
from scipy import stats

alpha = 0.05
confidence = 0.95

# 标准正态临界值
z_crit = stats.norm.ppf(1 - alpha / 2)
print(f"z_{{{alpha/2}}} = {z_crit:.4f}")

# 不同自由度下的t临界值
for df in [5, 10, 30, 100]:
    t_crit = stats.t.ppf(1 - alpha / 2, df)
    print(f"t_{{{alpha/2}}}({df}) = {t_crit:.4f}")
```

输出：
```
z_0.025 = 1.9600
t_0.025(5)   = 2.5706
t_0.025(10)  = 2.2281
t_0.025(30)  = 2.0423
t_0.025(100) = 1.9840
```

## Python代码示例

### 示例1：均数的置信区间

```python
import numpy as np
from scipy import stats

# 模拟数据：某药物的降血压效果（mmHg）
np.random.seed(42)
data = np.random.normal(loc=12.5, scale=3.0, size=30)

n = len(data)
xbar = np.mean(data)
s = np.std(data, ddof=1)
se = s / np.sqrt(n)

# 手动计算
t_crit = stats.t.ppf(0.975, df=n-1)
ci_manual = (xbar - t_crit * se, xbar + t_crit * se)
print(f"样本均值: {xbar:.2f}")
print(f"手动计算 95% CI: ({ci_manual[0]:.2f}, {ci_manual[1]:.2f})")

# 使用 scipy.stats.t.interval
ci_interval = stats.t.interval(
    confidence=0.95,
    df=n-1,
    loc=xbar,
    scale=se
)
print(f"t.interval 95% CI: ({ci_interval[0]:.2f}, {ci_interval[1]:.2f})")

# 输出结果
# 样本均值: 12.39
# 手动计算 95% CI: (11.20, 13.57)
# t.interval 95% CI: (11.20, 13.57)
```

### 示例2：率的置信区间

```python
import numpy as np
from scipy import stats

# 某临床试验：120名患者中84人有效
n = 120
x = 84
p = x / n

# 正态近似法
z_crit = stats.norm.ppf(0.975)
se = np.sqrt(p * (1 - p) / n)
ci_normal = (p - z_crit * se, p + z_crit * se)
print(f"有效率: {p:.1%}")
print(f"正态近似 95% CI: ({ci_normal[0]:.1%}, {ci_normal[1]:.1%})")

# 精确法（Clopper-Pearson）
ci_exact = stats.binom.interval(0.95, n, p)
print(f"精确法 95% CI: ({ci_exact[0]/n:.1%}, {ci_exact[1]/n:.1%})")
```

## 正确理解95%置信区间

### 频率学派解释（正确）

> 如果从总体中重复抽取大量样本（每次样本量相同），对每个样本构造95%置信区间，那么**约有95%的区间**会包含真实的总体参数。

这是一个**过程**（procedure）的属性，而非单个区间的属性。

### 常见误解（错误）

| 错误说法 | 正确理解 |
|----------|----------|
| "总体参数有95%的概率落在该区间内" | 参数是固定值，区间是随机的，不能说概率 |
| "该区间覆盖了95%的样本数据" | 置信区间是关于参数，不是关于数据 |
| "两个95%置信区间重叠意味着无差异" | 区间重叠不直接等价于统计不显著 |

### 关键点总结

1. 置信水平是**长期频率**的概念
2. 单个区间要么包含真值，要么不包含 —— **不能**说"有95%的概率包含"
3. 区间宽度与样本量成反比：$n \uparrow$ 则宽度 $\downarrow$
4. 区间宽度与置信水平成正比：置信水平 $\uparrow$ 则宽度 $\uparrow$
