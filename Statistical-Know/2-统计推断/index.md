# 统计推断

从样本信息推断总体特征，是统计学的核心。

## 三大支柱

| 方法 | 解决的问题 | 章节来源 |
|------|-----------|---------|
| 参数估计 | 总体参数是多少？ | 医统第6章 / 生统第4章 |
| 假设检验 | 差异是否由随机引起？ | 医统第6-7章 / 生统第4章 |
| t检验 | 两组均值是否不同？ | 医统第7章 / 生统第4章 |

## 整体流程

```
样本数据 → 参数估计（点估计+区间估计）
       → 假设检验（建立H0→计算统计量→p值→结论）
       → t检验（单样本/独立/配对）
       → 样本量估算（设计阶段）
```
