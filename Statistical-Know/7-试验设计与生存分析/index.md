# 试验设计与生存分析

研究设计与时间事件数据的分析方法。

## 内容

### 试验设计（生物统计学第10章 / 医学统计学第15章）

| 设计类型 | 特点 | 适用场景 |
|---------|------|---------|
| 完全随机设计 | 最简单，随机分组 | 实验室实验 |
| 随机区组设计 | 控制已知混杂 | 动物实验 |
| 拉丁方设计 | 控制两个因素 | 多因素实验 |
| 交叉设计 | 自身对照 | 临床试验 |
| 析因设计 | 研究交互效应 | 多因素研究 |
| 裂区设计 | 不同精度因子 | 田间/工艺试验 |

### 生存分析（医学统计学第14章）

| 方法 | 用途 |
|------|------|
| KM曲线 | 估计生存函数 |
| Log-rank检验 | 比较生存曲线 |
| Cox模型 | 多因素生存分析 |
