# 完全随机设计与随机区组设计

## 完全随机设计 (Completely Randomized Design, CRD)

### 原理

完全随机设计是最基本的实验设计类型。将实验单位完全随机地分配到 $k$ 个处理组中，每个组接受不同的处理。随机化的目的是平衡各组中已知和未知的混杂因素。

**统计模型：**

$$Y_{ij} = \mu + \tau_i + \varepsilon_{ij}, \quad \varepsilon_{ij} \sim N(0, \sigma^2)$$

其中 $Y_{ij}$ 为第 $i$ 组第 $j$ 个观测值，$\mu$ 为总体均值，$\tau_i$ 为第 $i$ 个处理的效应，$\varepsilon_{ij}$ 为随机误差。

### 随机化方法

使用随机数表或计算机生成随机序列：

```python
import numpy as np
import pandas as pd

# 模拟完全随机分组：30只小鼠，分为3组
np.random.seed(42)
n = 30
groups = ['对照', '低剂量', '高剂量']

# 生成随机排列后分配
assignment = np.random.permutation(np.repeat(groups, n // len(groups)))
df_crd = pd.DataFrame({'小鼠编号': range(1, n+1), '分组': assignment})
print(df_crd.head(10))
```

### 优缺点

| 优点 | 缺点 |
|------|------|
| 设计简单，分析容易 | 无法控制已知混杂因素 |
| 自由度最大，统计效率高 | 组间基线可能不均衡 |
| 灵活增减处理组数量 | 异质性较大时误差方差大 |

## 随机区组设计 (Randomized Block Design, RBD)

### 原理

将实验单位按某种已知特征（如窝别、体重、性别）划分为同质的"区组"，在每个区组内随机分配各处理。区组变量被作为第二个因素纳入模型，从而将区组变异从误差中分离出来。

**统计模型：**

$$Y_{ij} = \mu + \tau_i + \beta_j + \varepsilon_{ij}, \quad \varepsilon_{ij} \sim N(0, \sigma^2)$$

其中 $\beta_j$ 为第 $j$ 个区组的效应。与 CRD 相比，RBD 的误差项中排除了区组变异，因此检验更敏感。

**区组内方差缩减示意图：**

$$\text{CRD的MSE} \approx \text{RBD的MSE} + \text{区组变异}$$

### 控制混杂的原理

区组化是最有效的控制已知混杂的方法之一。例如，动物实验中按窝别区组：同一窝小鼠遗传背景相似，在窝内进行比较可消除遗传差异的影响。

### 统计效率提升

当区组间差异显著时，RBD 比 CRD 有更小的标准误和更大的检验功效。**区组相对效率 (Relative Efficiency)：**

$$\text{RE} = \frac{\text{MSE}_{\text{CRD}}}{\text{MSE}_{\text{RBD}}}$$

若 $\text{RE} > 1$，说明区组化提升了效率。

### Python 模拟随机区组分组

```python
# 随机区组分组：4个区组，每组3个处理
np.random.seed(42)
blocks = ['Block1', 'Block2', 'Block3', 'Block4']
treatments = ['A', 'B', 'C']

assignments = []
for block in blocks:
    block_assign = np.random.permutation(treatments)
    for treat in block_assign:
        assignments.append({'区组': block, '处理': treat})

df_rbd = pd.DataFrame(assignments)
print(df_rbd)
```

## t 检验 / ANOVA 与设计类型的对应

| 设计类型 | 两组比较 | 多组比较 | 软件实现（Python） |
|---------|---------|---------|------------------|
| CRD | 两独立样本 t 检验 | 单因素 ANOVA | `scipy.stats.ttest_ind` / `statsmodels.OLS` |
| RBD | 配对 t 检验（配对设计） | 随机区组 ANOVA | `statsmodels.OLS` 加入区组因子 |

**关键概念：配对设计本质上是只有2个处理水平的RBD。**

> **要点总结：** 选择设计类型时，若已知重要的混杂因素，优先考虑随机区组设计；若实验单位高度同质或混杂因素未知，完全随机设计更为简单高效。
