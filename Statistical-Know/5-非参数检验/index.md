# 非参数检验

不依赖总体分布假设的统计方法，适用于偏态分布或等级数据。

## 内容

| 方法 | 对应参数检验 | 来源 |
|------|------------|------|
| 符号检验 | — | 生统第5章 |
| Wilcoxon符号秩检验 | 配对t检验 | 医统第10章 / 生统第5章 |
| Mann-Whitney U检验 | 独立样本t检验 | 生统第5章 |
| Kruskal-Wallis检验 | 单因素ANOVA | 医统第10章 / 生统第5章 |
| Friedman检验 | 重复测量ANOVA | 生统第5章 |

## 选择原则

```
总体正态 + 方差齐 → 参数检验（功效更高）
总体偏态 / 等级数据 → 非参数检验（更稳健）
小样本 + 非正态 → 非参数检验（更安全）
```
