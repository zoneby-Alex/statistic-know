# 知识路线导览

生物医学统计知识体系的全局路线，展示知识点之间的逻辑脉络。

## 核心知识路线

```
┌─────────────────────────────────────────────────────────────┐
│                    生物医学统计知识体系                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【统计基础】                                                │
│  概率论 → 抽样分布 → 正态分布                                │
│  → 定量描述（均值/方差） → 定性描述（率/构成比）              │
│                                                             │
│  【统计推断】                                                │
│  参数估计（点估计/区间估计） → 假设检验（p值/两类错误）       │
│  → t检验（单样本/独立/配对） → 样本量估算                    │
│                                                             │
│  【多组比较】                                                │
│  方差分析（单因素/多因素/重复测量） → 多重比较 → 协方差分析   │
│                                                             │
│  【分类与等级】                                              │
│  卡方检验（拟合优度/独立性） → Fisher/McNemar → 非参数检验    │
│                                                             │
│  【回归与相关】                                              │
│  直线回归与相关 → 多元回归 → Logistic回归 → 通径分析         │
│                                                             │
│  【设计与生存】                                              │
│  试验设计（CRD/RBD/拉丁方/析因） → 生存分析（KM/Cox）        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 独立学习路径

按生物医学研究流程组织的推荐学习顺序：

```
研究设计 → 试验设计（第7章试验设计模块）
    ↓
数据收集 → 数据类型判断（定量/定性/时间事件）
    ↓
数据描述 → 统计表与统计图（第1章统计基础模块）
    ↓
统计分析 → 根据研究问题选择方法
    ├── 两组比较 → t检验 / Mann-Whitney检验
    ├── 多组比较 → 方差分析 / Kruskal-Wallis检验
    ├── 分类关联 → 卡方检验 / Fisher检验
    ├── 影响因素 → 回归分析（直线/多元/Logistic）
    ├── 时间事件 → 生存分析（KM/Cox）
    └── 控制混杂 → 协方差分析 / 分层分析
    ↓
结果解释 → 统计报告
```
