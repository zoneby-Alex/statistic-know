# 重复测量方差分析（Repeated Measures ANOVA）

## 核心概念

对**同一组对象在不同时间点或多个条件下**进行多次测量，分析测量值随时间或条件的变化。

### 适用场景

| 设计类型 | 示例 |
|----------|------|
| 时间序列 | 治疗前、治疗1个月、治疗3个月的血压 |
| 条件重复 | 同一组患者在三种不同药物下的反应 |
| 双侧对照 | 同一患者的左眼和右眼接受不同治疗 |

### 与单因素ANOVA的区别

| 特征 | 重复测量ANOVA | 单因素ANOVA |
|------|--------------|-------------|
| 数据关系 | 同一对象多次测量（相关） | 不同对象（独立） |
| 变异分解 | 可分离出**个体间变异** | 个体变异混入组内误差 |
| 统计功效 | 更高（控制了个体差异） | 较低 |
| 自由度 | 需要校正（球形假设） | 标准 |

---

## 变异分解

重复测量ANOVA将总变异分解为：

$$
SS_{\text{总}} = SS_{\text{个体间}} + SS_{\text{时间（处理）}} + SS_{\text{误差}}
$$

其中 $SS_{\text{个体间}}$ 反映不同研究对象之间的固有差异，将其分离出去后，**误差项减小**，从而提高了对时间效应的检验功效。

### F统计量

$$
F_{\text{时间}} = \frac{MS_{\text{时间}}}{MS_{\text{误差}}}
$$

---

## 球形假设（Sphericity Assumption）

### 定义

球形假设要求任何两个时间点测量值的**差值方差相等**。即：

$$
\sigma^2_{(Y_i - Y_j)} = \sigma^2_{(Y_k - Y_l)} \quad \text{对所有 } i \neq j, k \neq l \text{ 成立}
$$

### Mauchly球形检验

- $H_0$：满足球形假设
- $H_1$：不满足球形假设

### 不满足时的校正方法

| 校正方法 | epsilon ($\varepsilon$) | 适用情况 |
|----------|----------------------|----------|
| **Greenhouse-Geisser** | 最保守（下限） | $\varepsilon$ 较小时 |
| **Huynh-Feldt** | 较宽松 | $\varepsilon$ 接近1时 |
| **Lower-bound** | 最保守 | 极端情况 |

校正后的自由度乘以 $\varepsilon$，$p$ 值相应增大（更保守）。

## Python代码示例

### 使用 statsmodels AnovaRM

```python
import pandas as pd
import numpy as np
from statsmodels.stats.anova import AnovaRM
import matplotlib.pyplot as plt

# 模拟数据：10名患者在4个时间点的焦虑评分
np.random.seed(42)

n_subjects = 10
time_points = 4

# 生成数据：随时间推移焦虑水平下降
data = []
for subject in range(1, n_subjects + 1):
    baseline = np.random.normal(70, 8)  # 个体基线水平
    for tp in range(time_points):
        # 时间效应：线性下降 + 个体变异
        time_effect = -5 * tp
        score = baseline + time_effect + np.random.normal(0, 5)
        data.append({
            'subject': subject,
            'time': f'T{tp+1}',  # T1, T2, T3, T4
            'score': score
        })

df = pd.DataFrame(data)

# 描述性统计
print("各时间点描述性统计:")
print(df.groupby('time')['score'].describe())
```

### 重复测量ANOVA

```python
# 重复测量ANOVA
rm_anova = AnovaRM(
    data=df,
    depvar='score',
    subject='subject',
    within=['time']
)
result = rm_anova.fit()

print("\n重复测量ANOVA结果:")
print(result.anova_table)
```

### 手动计算关键统计量

```python
# 个体均值（去除时间效应）
subject_means = df.groupby('subject')['score'].mean()
grand_mean = df['score'].mean()

# 时间点均值
time_means = df.groupby('time')['score'].mean()

# 总变异
ss_total = sum((df['score'] - grand_mean)**2)

# 个体间变异
ss_between_subjects = sum([
    (sub_mean - grand_mean)**2 * time_points
    for sub_mean in subject_means
])

# 时间效应
ss_time = sum([
    (tm - grand_mean)**2 * n_subjects
    for tm in time_means
])

# 残差
ss_error = ss_total - ss_between_subjects - ss_time

print(f"SS_total = {ss_total:.2f}")
print(f"SS_between_subjects = {ss_between_subjects:.2f}")
print(f"SS_time = {ss_time:.2f}")
print(f"SS_error = {ss_error:.2f}")

# F统计量
df_time = time_points - 1
df_error = (n_subjects - 1) * (time_points - 1)
ms_time = ss_time / df_time
ms_error = ss_error / df_error
f_stat = ms_time / ms_error
print(f"\nF({df_time}, {df_error}) = {f_stat:.3f}")
```

### 球形检验与校正

```python
from statsmodels.stats.anova import AnovaRM
# AnovaRM内部会进行Mauchly球形检验并提供校正结果

# 获取Mauchly检验结果（statsmodels较新版本支持）
print("\n注意：AnovaRM的球形检验依赖于statsmodels版本")
print("可以使用pingouin库进行更完整的球形检验")

# 使用pingouin进行更完整的重复测量方差分析
try:
    import pingouin as pg
    aov = pg.rm_anova(
        data=df,
        dv='score',
        within='time',
        subject='subject',
        detailed=True,
        correction=True  # 自动应用Greenhouse-Geisser校正
    )
    print("\nPingouin重复测量ANOVA结果:")
    print(aov)
    
    # 球形检验
    spher = pg.sphericity(data=df, dv='score', subject='subject', within='time')
    print(f"\nMauchly球形检验:")
    print(f"  W = {spher.sphericity_test['W'].values[0]:.4f}")
    print(f"  p = {spher.sphericity_test['pval'].values[0]:.4f}")
    print(f"  球形假设{'满足' if spher.sphericity_test['sphericity'].values[0] else '不满足'}")
    if not spher.sphericity_test['sphericity'].values[0]:
        eps = spher.sphericity_test['eps'].values[0]
        print(f"  Greenhouse-Geisser epsilon = {eps:.4f}")
except ImportError:
    print("pingouin未安装，可以运行: pip install pingouin")
```

### 事后比较

```python
from scipy import stats

# 重复测量数据的两两比较（使用配对t检验 + Bonferroni校正）
time_vals = ['T1', 'T2', 'T3', 'T4']
pairs = []
p_values = []

for i in range(len(time_vals)):
    for j in range(i+1, len(time_vals)):
        t1_data = df[df['time'] == time_vals[i]]['score'].values
        t2_data = df[df['time'] == time_vals[j]]['score'].values
        _, p = stats.ttest_rel(t1_data, t2_data)
        pairs.append(f"{time_vals[i]} vs {time_vals[j]}")
        p_values.append(p)

# Bonferroni校正
from statsmodels.stats.multitest import multipletests
reject, p_corrected, _, _ = multipletests(p_values, method='bonferroni')

print("\n事后比较（Bonferroni校正）:")
for pair, p_raw, p_corr, rej in zip(pairs, p_values, p_corrected, reject):
    print(f"  {pair}: 原始p={p_raw:.4f}, 校正p={p_corr:.4f}, {'显著' if rej else '不显著'}")
```

## 与混合效应模型的对比

### 重复测量ANOVA的局限

1. **严格的球形假设**（实际数据很难满足）
2. **无法处理缺失值**（必须删除整个个案）
3. **要求平衡设计**（所有对象测量次数相同）
4. **仅适用于连续型因变量**

### 混合效应模型（LMM）的优势

| 特性 | 重复测量ANOVA | 混合效应模型 |
|------|--------------|-------------|
| 缺失值 | 无法处理 | **可处理** |
| 球形假设 | 要求 | **不要求**（灵活协方差结构） |
| 不平衡数据 | 无法处理 | **可处理** |
| 时间变量 | 分类（离散） | 连续或分类均可 |
| 随机效应 | 仅随机截距 | 随机截距+随机斜率 |

### Python中的混合效应模型

```python
import statsmodels.api as sm
from statsmodels.formula.api import mixedlm

# 使用混合效应模型分析重复测量数据
# 将时间作为连续变量，subject作为随机截距
df['time_num'] = df['time'].str.extract('(\d+)').astype(int)

lmm_model = mixedlm(
    "score ~ time_num",      # 固定效应：时间的线性趋势
    data=df,
    groups=df["subject"],     # 随机效应：个体随机截距
    re_formula="~time_num"    # 随机斜率：个体随机斜率
).fit()

print("\n混合效应模型结果:")
print(lmm_model.summary())
```

## 结果报告模板

### 中文报告

> 采用单因素重复测量方差分析考察10名患者在4个时间点（T1-T4）的焦虑评分变化。Mauchly球形检验结果满足球形假设（$W = 0.524, p = 0.612$）。时间主效应显著（$F(3, 27) = 28.67, p < 0.001, \eta_p^2 = 0.761$），表明不同时间点的焦虑评分存在统计学差异。事后比较（Bonferroni校正）显示，T1与T2、T2与T3、T3与T4之间差异均显著（校正后 $p < 0.05$），焦虑评分随时间呈线性下降趋势。

---

## 关键点总结

1. 重复测量ANOVA通过**分离个体间变异**来提高统计功效
2. **球形假设**是核心前提，不满足时需使用Greenhouse-Geisser校正
3. 相比独立组设计，重复测量设计**节省样本量**（同一对象多次测量）
4. 缺失数据较多或球形假设严重违反时，考虑使用**混合效应模型**
