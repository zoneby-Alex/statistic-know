# 单因素方差分析（One-Way ANOVA）

## 核心概念

比较**三个及以上**独立组的均值是否存在统计学差异。

### 适用场景

- 四种不同药物的疗效比较
- 三个不同年龄组的BMI差异
- 五种不同温度下酶活性比较

### 前提条件

| 条件 | 检验方法 |
|------|----------|
| 各组数据独立 | 研究设计保证 |
| 各组服从正态分布 | Shapiro-Wilk或K-S检验 |
| 各组方差齐性 | Levene检验或Bartlett检验 |

---

## 变异分解原理

ANOVA的核心思想是将总变异分解为两部分：

$$
SS_{\text{总}} = SS_{\text{组间}} + SS_{\text{组内}}
$$

### 公式详解

**总离差平方和：**

$$
SS_{\text{总}} = \sum_{i=1}^{k}\sum_{j=1}^{n_i} (x_{ij} - \bar{x})^2
$$

**组间离差平方和（处理效应 + 随机误差）：**

$$
SS_{\text{组间}} = \sum_{i=1}^{k} n_i (\bar{x}_i - \bar{x})^2
$$

**组内离差平方和（随机误差）：**

$$
SS_{\text{组内}} = \sum_{i=1}^{k}\sum_{j=1}^{n_i} (x_{ij} - \bar{x}_i)^2
$$

### 均方与F统计量

$$
MS_{\text{组间}} = \frac{SS_{\text{组间}}}{k-1}, \quad
MS_{\text{组内}} = \frac{SS_{\text{组内}}}{N-k}
$$

$$
F = \frac{MS_{\text{组间}}}{MS_{\text{组内}}} \sim F(k-1,\; N-k)
$$

其中 $k$ 为组数，$N$ 为总样本量，$n_i$ 为第 $i$ 组的样本量。

---

## ANOVA表解读

| 变异来源 | SS | df | MS | F | p |
|----------|-----|----|-----|-----|-----|
| 组间 | 120.5 | 2 | 60.25 | 8.42 | 0.001 |
| 组内 | 128.8 | 18 | 7.16 | | |
| 总计 | 249.3 | 20 | | | |

**关键解读：**
- $F$ 值越大，组间差异相对于组内变异越显著
- $p < 0.05$ 表示至少有一组与其他组存在显著差异
- **ANOVA显著 ≠ 所有组间两两差异显著**（需事后比较）

## Python代码示例

### 使用 scipy.stats.f_oneway

```python
import numpy as np
from scipy import stats

# 示例：三种不同饮食方案对体重减轻的效果（kg）
np.random.seed(42)
diet_a = np.random.normal(loc=3.5, scale=1.2, size=20)  # 低碳水
diet_b = np.random.normal(loc=4.8, scale=1.3, size=20)  # 地中海
diet_c = np.random.normal(loc=2.9, scale=1.1, size=20)  # 低脂

# 描述性统计
for name, data in [('低碳水饮食', diet_a), ('地中海饮食', diet_b), ('低脂饮食', diet_c)]:
    print(f"{name}: n={len(data)}, M={np.mean(data):.2f}, SD={np.std(data, ddof=1):.2f}")

# 单因素方差分析
f_stat, p_value = stats.f_oneway(diet_a, diet_b, diet_c)
print(f"\n单因素ANOVA结果:")
print(f"  F({2}, {57}) = {f_stat:.4f}, p = {p_value:.6f}")

if p_value < 0.05:
    print("  结论：至少有一种饮食方案的减重效果与其他方案存在显著差异")
else:
    print("  结论：三种饮食方案的减重效果无显著差异")
```

### 完整ANOVA分析（使用statsmodels）

```python
import pandas as pd
import numpy as np
import statsmodels.api as sm
from statsmodels.formula.api import ols
from scipy import stats

# 构建长格式数据
df = pd.DataFrame({
    'weight_loss': np.concatenate([diet_a, diet_b, diet_c]),
    'diet': ['低碳水'] * 20 + ['地中海'] * 20 + ['低脂'] * 20
})

# 进行ANOVA
model = ols('weight_loss ~ C(diet)', data=df).fit()
anova_table = sm.stats.anova_lm(model, typ=2)
print("ANOVA分析表（Type II）:")
print(anova_table)

# 模型摘要
print("\n模型摘要:")
print(model.summary())
```

## 事后比较（Post-hoc Tests）

ANOVA显著只能告诉我们"至少有一组不同"，但不知道具体哪些组不同，因此需要事后比较。

### Tukey HSD（Honestly Significant Difference）

最常用的事后比较方法，控制了**多重比较的族系错误率（FWER）**。

$$
\text{HSD} = q_{\alpha}(k, df_{\text{组内}}) \cdot \sqrt{\frac{MS_{\text{组内}}}{n}}
$$

```python
from statsmodels.stats.multicomp import pairwise_tukeyhsd
import matplotlib.pyplot as plt

# Tukey HSD事后比较
tukey = pairwise_tukeyhsd(
    endog=df['weight_loss'],
    groups=df['diet'],
    alpha=0.05
)

print("Tukey HSD事后比较结果:")
print(tukey)

# 可视化
fig = tukey.plot_simultaneous()
plt.title('Tukey HSD 多重比较')
plt.xlabel('均值差')
plt.tight_layout()
plt.show()
```

### 事后比较方法选择

| 方法 | 适用场景 | 控制水平 |
|------|----------|----------|
| **Tukey HSD** | 所有组两两比较 | FWER |
| **Bonferroni** | 少数指定的比较 | FWER（最保守） |
| **Dunnett** | 多组与对照组的比较 | FWER |
| **Scheffe** | 复杂对比（如合并组） | FWER |
| **FDR校正** | 探索性分析 | FDR（较宽松） |

```python
from statsmodels.stats.multitest import multipletests
from scipy import stats

# 手动进行两两比较并用Bonferroni校正
groups = [diet_a, diet_b, diet_c]
group_names = ['低碳水', '地中海', '低脂']
pairs = [(0, 1), (0, 2), (1, 2)]

p_values = []
for i, j in pairs:
    _, p = stats.ttest_ind(groups[i], groups[j], equal_var=True)
    p_values.append(p)

# Bonferroni校正
reject, p_corrected, _, _ = multipletests(p_values, method='bonferroni')

print("Bonferroni校正后结果:")
for (i, j), p_raw, p_corr, rej in zip(pairs, p_values, p_corrected, reject):
    print(f"  {group_names[i]} vs {group_names[j]}: 原始p={p_raw:.4f}, 校正p={p_corr:.4f}, {'显著' if rej else '不显著'}")
```

## 效应量：Eta-squared ($\eta^2$)

$$
\eta^2 = \frac{SS_{\text{组间}}}{SS_{\text{总}}}
$$

解释力度：$\eta^2$ 表示组别所能解释的变异占总变异的比例。

| $\eta^2$ | 效应大小 |
|----------|----------|
| 0.01 | 小 |
| 0.06 | 中 |
| 0.14 | 大 |

```python
# 手动计算eta-squared
ss_between = sum([
    len(data) * (np.mean(data) - np.mean(np.concatenate([diet_a, diet_b, diet_c])))**2
    for data in [diet_a, diet_b, diet_c]
])
ss_total = sum([
    sum((x - np.mean(np.concatenate([diet_a, diet_b, diet_c])))**2)
    for data in [diet_a, diet_b, diet_c] for x in data
])
eta_sq = ss_between / ss_total
print(f"η² = {eta_sq:.3f}")
```

## ANOVA的假设检验流程

```python
def oneway_anova_full(*groups, group_names=None, alpha=0.05):
    """单因素方差分析完整流程"""
    import warnings
    warnings.filterwarnings('ignore')
    
    if group_names is None:
        group_names = [f'组{i+1}' for i in range(len(groups))]
    
    print("=" * 60)
    print("单因素方差分析完整报告")
    print("=" * 60)
    
    # 1. 描述性统计
    print("\n1. 各组描述性统计:")
    all_data = np.concatenate(groups)
    for name, data in zip(group_names, groups):
        print(f"   {name}: n={len(data)}, M={np.mean(data):.2f}, SD={np.std(data, ddof=1):.2f}")
    
    # 2. 正态性检验
    print("\n2. 正态性检验 (Shapiro-Wilk):")
    all_normal = True
    for name, data in zip(group_names, groups):
        _, p = stats.shapiro(data)
        result = "通过" if p > alpha else "偏离"
        if p <= alpha:
            all_normal = False
        print(f"   {name}: W = ..., p = {p:.4f} ({result})")
    
    # 3. 方差齐性检验
    print("\n3. 方差齐性检验 (Levene):")
    _, p_levene = stats.levene(*groups)
    equal_var = p_levene > alpha
    print(f"   Levene统计量: p = {p_levene:.4f} {'(通过)' if equal_var else '(不齐，建议Welch ANOVA)'}")
    
    # 4. ANOVA
    print("\n4. 方差分析结果:")
    if equal_var:
        f_stat, p_val = stats.f_oneway(*groups)
        k = len(groups)
        n_total = sum(len(g) for g in groups)
        print(f"   F({k-1}, {n_total-k}) = {f_stat:.4f}, p = {p_val:.6f}")
    else:
        # Welch ANOVA
        from scipy.stats import kruskal
        print("   方差齐性不满足，建议使用Welch ANOVA或Kruskal-Wallis非参数检验")
        h_stat, p_val = kruskal(*groups)
        print(f"   Kruskal-Wallis: H = {h_stat:.4f}, p = {p_val:.6f}")
    
    # 5. 结果判断
    print(f"\n5. 结论:")
    if p_val < alpha:
        print(f"   各组间存在统计学差异 (p = {p_val:.4f})")
        if equal_var and len(groups) > 2:
            print("\n6. 事后比较 (Tukey HSD):")
            import pandas as pd
            df = pd.DataFrame({
                'value': all_data,
                'group': np.repeat(group_names, [len(g) for g in groups])
            })
            tukey = pairwise_tukeyhsd(df['value'], df['group'], alpha=alpha)
            print(tukey)
    else:
        print(f"   各组间差异无统计学意义 (p = {p_val:.4f})")

# 运行完整分析
oneway_anova_full(diet_a, diet_b, diet_c,
                  group_names=['低碳水', '地中海', '低脂'])
```

## 关键点总结

1. ANOVA比较的是**均值差异**，但基于**方差分解**。理解 $SS$ 分解是掌握ANOVA的关键。
2. ANOVA显著不代表所有组间两两比较都显著，必须做事后比较。
3. 方差齐性不满足时，考虑Welch ANOVA或Kruskal-Wallis检验。
4. 务必报告效应量 $\eta^2$，$p$ 值不能反映差异大小。
