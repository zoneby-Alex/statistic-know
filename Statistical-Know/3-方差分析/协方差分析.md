# 协方差分析（ANCOVA）

## 核心概念

在比较组间均值差异时，**控制一个或多个连续型协变量的影响**，从而减少误差变异，提高检验功效。

### 何时使用ANCOVA

| 场景 | 说明 |
|------|------|
| 基线值校正 | 比较治疗组和对照组的终点值，控制基线值差异 |
| 混杂变量控制 | 各组在某连续变量上不均衡，需统计调整 |
| 提高统计功效 | 协变量可解释部分变异，减少误差项 |

### ANCOVA的本质

$$
\text{ANCOVA} = \text{ANOVA} + \text{回归}
$$

在剔除协变量的影响后，比较各组的**调整后均值**。

---

## ANCOVA的原理

### 模型方程

$$
y_{ij} = \mu + \alpha_i + \beta(x_{ij} - \bar{x}) + \varepsilon_{ij}
$$

其中：
- $y_{ij}$：第 $i$ 组第 $j$ 个观察对象的因变量值
- $\mu$：总体均值
- $\alpha_i$：第 $i$ 组的效应（处理效应）
- $\beta$：协变量 $x$ 的回归系数
- $x_{ij}$：第 $i$ 组第 $j$ 个观察对象的协变量值
- $\bar{x}$：协变量的总均值
- $\varepsilon_{ij}$：随机误差

### 变异分解

$$
SS_{\text{总}} = SS_{\text{处理（组别）}} + SS_{\text{协变量}} + SS_{\text{误差}}
$$

协变量解释的部分被从误差项中分离出去，使得 $MS_{\text{误差}}$ 减小，$F$ 值增大。

---

## 调整后均值

调整后均值是在除去协变量影响后，各组因变量的估计均值。

### 计算公式

$$
\bar{y}_{i(\text{adj})} = \bar{y}_i - \beta(\bar{x}_i - \bar{x})
$$

其中 $\beta$ 为公共回归斜率（假设各组斜率齐性）。

**直观理解：**
- 如果第 $i$ 组的协变量均值高于总均值（$\bar{x}_i > \bar{x}$），则调整后均值会降低
- 如果第 $i$ 组的协变量均值低于总均值（$\bar{x}_i < \bar{x}$），则调整后均值会升高

---

## 斜率齐性假设

### 定义

各组中协变量与因变量的回归斜率应该相同。

$$
H_0: \beta_1 = \beta_2 = \cdots = \beta_k
$$

### 检验方法

检验交互项 `组别 × 协变量` 是否显著。

```python
# 斜率齐性检验（在拟合完整ANCOVA模型之前）
# 如果 组别:协变量 的交互项显著，说明斜率不齐
```

### 斜率不齐时的处理方法

| 方法 | 说明 |
|------|------|
| 加入交互项 | 使用含交互项的模型（Johnson-Neyman法） |
| 分组分析 | 分别在各组内做回归分析 |
| 非参数方法 | 使用匹配或分层方法 |

## Python代码示例

### 使用 statsmodels ols 进行ANCOVA

```python
import numpy as np
import pandas as pd
import statsmodels.api as sm
from statsmodels.formula.api import ols
import matplotlib.pyplot as plt

# 模拟数据：评估两种教学方法的效果，控制前测成绩
np.random.seed(42)

n_per_group = 30

# 模拟前测成绩（协变量）
pretest_A = np.random.normal(65, 10, n_per_group)
pretest_B = np.random.normal(70, 10, n_per_group)  # B组前测略高

# 模拟后测成绩（因变量）
# 假设：方法A平均增幅5分，方法B平均增幅10分
posttest_A = pretest_A * 0.7 + 20 + np.random.normal(5, 5, n_per_group)
posttest_B = pretest_B * 0.7 + 20 + np.random.normal(10, 5, n_per_group)

# 构建数据框
df = pd.DataFrame({
    'pretest': np.concatenate([pretest_A, pretest_B]),
    'posttest': np.concatenate([posttest_A, posttest_B]),
    'method': ['传统教学'] * n_per_group + ['新教学法'] * n_per_group
})

print("各组描述性统计（未调整）:")
print(df.groupby('method').agg({
    'pretest': ['mean', 'std'],
    'posttest': ['mean', 'std']
}))
```

### 基本的ANCOVA分析

```python
# ANCOVA模型：控制前测成绩，比较两种教学方法
# 公式: 因变量 ~ 协变量 + 分组变量
model_ancova = ols('posttest ~ pretest + C(method)', data=df).fit()
anova_result = sm.stats.anova_lm(model_ancova, typ=2)

print("\nANCOVA结果:")
print(anova_result)
print()

# 与未控制协变量的ANOVA对比
model_anova = ols('posttest ~ C(method)', data=df).fit()
anova_only = sm.stats.anova_lm(model_anova, typ=2)

print("简单ANOVA结果（未控制协变量）:")
print(anova_only)
```

### 调整后均值的计算

```python
from statsmodels.stats.multicomp import pairwise_tukeyhsd

# 获取调整后均值
# 方法：对每个组的协变量取均值后预测
grand_mean_pretest = df['pretest'].mean()

# 预测调整后均值：将协变量设为总均值
print(f"\n协变量总均值: {grand_mean_pretest:.2f}")

# 创建预测数据框：各组 + 协变量固定为总均值
predict_df = pd.DataFrame({
    'method': ['传统教学', '新教学法'],
    'pretest': [grand_mean_pretest, grand_mean_pretest]
})

# 添加预测值及置信区间
predictions = model_ancova.get_prediction(predict_df)
pred_summary = predictions.summary_frame(alpha=0.05)

print("\n调整后均值:")
for i, method in enumerate(['传统教学', '新教学法']):
    print(f"  {method}: {pred_summary.loc[i, 'mean']:.2f} "
          f"({pred_summary.loc[i, 'mean_ci_lower']:.2f}, "
          f"{pred_summary.loc[i, 'mean_ci_upper']:.2f})")

# 原始均值 vs 调整后均值对比
raw_means = df.groupby('method')['posttest'].mean()
print(f"\n原始均值:")
for method in ['传统教学', '新教学法']:
    print(f"  {method}: {raw_means[method]:.2f}")
```

### 斜率齐性检验

```python
# 斜率齐性检验：检验 组别 × 协变量 交互项是否显著
# 如果交互项显著，则斜率不齐，ANCOVA结果不可靠
model_interaction = ols('posttest ~ pretest * C(method)', data=df).fit()
interaction_anova = sm.stats.anova_lm(model_interaction, typ=2)

print("斜率齐性检验（交互项检验）:")
print(interaction_anova)

# 提取交互项的p值
interaction_p = interaction_anova.loc['pretest:C(method)', 'PR(>F)']
if interaction_p > 0.05:
    print(f"\n交互项p = {interaction_p:.4f} > 0.05，斜率齐性假设成立")
    print("ANCOVA结果可靠")
else:
    print(f"\n交互项p = {interaction_p:.4f} < 0.05，斜率不齐")
    print("建议使用Johnson-Neyman法或分组分析")
```

### 完整ANCOVA分析流程

```python
def ancova_full_analysis(df, dv, covariate, group, alpha=0.05):
    """ANCOVA完整分析流程"""
    print("=" * 60)
    print("协方差分析（ANCOVA）完整报告")
    print("=" * 60)
    
    # 1. 描述性统计
    print(f"\n1. 各组描述性统计:")
    print(df.groupby(group)[[dv, covariate]].describe().round(2))
    
    # 2. 斜率齐性检验
    print("\n2. 斜率齐性检验:")
    model_interact = ols(f'{dv} ~ {covariate} * C({group})', data=df).fit()
    anova_interact = sm.stats.anova_lm(model_interact, typ=2)
    p_slope = anova_interact.loc[f'C({group}):{covariate}', 'PR(>F)']
    print(f"   交互项 p = {p_slope:.4f}")
    
    if p_slope <= alpha:
        print("   斜率不齐！ANCOVA结果可能不可靠")
        print("   建议：使用Johnson-Neyman法或分组回归")
        return None
    
    print("   斜率齐性假设成立 ✓\n")
    
    # 3. ANCOVA
    print("3. ANCOVA结果:")
    model = ols(f'{dv} ~ {covariate} + C({group})', data=df).fit()
    anova_table = sm.stats.anova_lm(model, typ=2)
    print(anova_table)
    
    # 4. 调整后均值
    print("\n4. 调整后均值:")
    grand_mean = df[covariate].mean()
    pred_df = pd.DataFrame({
        group: df[group].unique(),
        covariate: [grand_mean] * len(df[group].unique())
    })
    preds = model.get_prediction(pred_df).summary_frame(alpha=alpha)
    
    for i, grp in enumerate(df[group].unique()):
        print(f"   {grp}: {preds.loc[i, 'mean']:.2f} "
              f"(95% CI: {preds.loc[i, 'mean_ci_lower']:.2f}, "
              f"{preds.loc[i, 'mean_ci_upper']:.2f})")
    
    # 5. 效应量（偏eta-squared）
    print("\n5. 效应量:")
    ss_group = anova_table.loc[f'C({group})', 'sum_sq']
    ss_error = anova_table.loc['Residual', 'sum_sq']
    eta_p = ss_group / (ss_group + ss_error)
    print(f"   偏η² = {eta_p:.3f}")
    
    # 6. 模型摘要
    print(f"\n6. 模型R²: {model.rsquared:.4f}")
    
    return model

# 运行完整分析
result = ancova_full_analysis(df, 'posttest', 'pretest', 'method')
```

## ANCOVA的假设总结

| 假设 | 检验方法 | 违反后果 |
|------|----------|----------|
| 协变量为连续变量 | 数据检查 | 应使用ANOVA |
| 协变量测量无误差 | 研究设计保证 | 回归稀释偏差 |
| 线性关系 | 散点图检查 | 可考虑非线性变换 |
| **斜率齐性** | 交互项检验 | ANCOVA结果偏差 |
| 协变量独立于处理 | 研究设计/随机化 | 调整偏差 |
| 因变量正态分布 | 残差分析 | 可考虑非参数方法 |
| 方差齐性 | Levene检验 | 使用稳健标准误 |

## 关键点总结

1. ANCOVA的本质是**在ANOVA框架中纳入回归分析**，控制协变量后的组间比较
2. **斜率齐性**是ANCOVA的核心前提，务必在正式分析前检验
3. 调整后均值是"如果各组协变量相同"时的估计均值
4. ANCOVA可以**显著提高统计功效**（协变量解释的变异不再计入误差）
5. 协变量应在**研究设计阶段确定**，而非事后从多个变量中挑选
