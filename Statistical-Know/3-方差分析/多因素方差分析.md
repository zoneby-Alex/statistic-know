# 多因素方差分析（Factorial ANOVA）

## 核心概念

同时考察**两个及以上自变量（因素）** 对因变量的影响，并分析因素间的**交互作用**。

### 典型应用

- 药物类型（新药 vs 安慰剂）与剂量（高 vs 低）对疗效的影响
- 性别（男 vs 女）与教学方法（传统 vs 创新）对成绩的影响
- 温度（三个水平）与pH值（两个水平）对酶活性的影响

### 优势

1. **效率更高**：一次试验考察多个因素
2. **交互效应**：揭示因素间的协同或拮抗作用（这是单因素ANOVA无法做到的）
3. **外部效度**：更接近真实世界（多因素同时变化）

---

## 主效应与交互效应

### 主效应（Main Effect）

某个因素在不同水平下对因变量的**平均影响**。

例如：考察性别和训练方法对跑步成绩的影响
- 性别的主效应：男女平均成绩是否不同
- 训练方法的主效应：不同训练方法的平均效果是否不同

### 交互效应（Interaction Effect）

一个因素的效果依赖于另一个因素的水平。

- **无交互**：各因素的效果可加（独立作用）
- **有交互**：因素A的效果在因素B的不同水平上不一致

### 变异分解（两因素设计）

$$
SS_{\text{总}} = SS_A + SS_B + SS_{A \times B} + SS_{\text{组内（误差）}}
$$

### F统计量

$$
F_A = \frac{MS_A}{MS_{\text{误差}}}, \quad
F_B = \frac{MS_B}{MS_{\text{误差}}}, \quad
F_{A \times B} = \frac{MS_{A \times B}}{MS_{\text{误差}}}
$$

## 交互作用图解读

### 交互作用类型

```
无交互（平行）        有交互（交叉）        有交互（非交叉）
   |  A2               |  A2                  |   A2
   |—— ———             |  ——                  |  ——
   |  A1   ——          | /   —— A1            | /  A1
   |—— ———  ——         |/       ——            |/   ——
   |________________   |________________      |________________
       B1   B2            B1   B2                B1   B2
```

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import statsmodels.api as sm
from statsmodels.formula.api import ols

# 模拟数据：药物类型 × 剂量
np.random.seed(42)

# 因素1：药物类型 (A, B)
# 因素2：剂量 (低, 中, 高)
drugs = ['药物A', '药物B']
doses = ['低剂量', '中剂量', '高剂量']

# 设计：2 × 3 完全随机设计，每组10个样本
data = []
for drug in drugs:
    for dose in doses:
        if drug == '药物A':
            if dose == '低剂量':
                values = np.random.normal(5, 1.5, 10)
            elif dose == '中剂量':
                values = np.random.normal(8, 1.5, 10)
            else:
                values = np.random.normal(12, 1.5, 10)
        else:  # 药物B
            if dose == '低剂量':
                values = np.random.normal(6, 1.5, 10)
            elif dose == '中剂量':
                values = np.random.normal(7, 1.5, 10)
            else:
                values = np.random.normal(8, 1.5, 10)
        for v in values:
            data.append({'drug': drug, 'dose': dose, 'effect': v})

df = pd.DataFrame(data)

# 2 × 3 设计的均值表
pivot = df.pivot_table(values='effect', index='drug', columns='dose', aggfunc='mean')
print("各组均值:")
print(pivot)
```

### 交互作用图

```python
# 计算均值和标准误
means = df.groupby(['drug', 'dose'])['effect'].agg(['mean', 'sem']).reset_index()

fig, ax = plt.subplots(figsize=(8, 6))

# 绘制交互作用图
dose_order = ['低剂量', '中剂量', '高剂量']
for drug in drugs:
    subset = means[means['drug'] == drug]
    x_pos = [dose_order.index(d) for d in subset['dose']]
    ax.errorbar(x_pos, subset['mean'], yerr=subset['sem'],
                marker='o', label=drug, capsize=5, capthick=2, linewidth=2)

ax.set_xticks(range(len(dose_order)))
ax.set_xticklabels(dose_order)
ax.set_xlabel('剂量')
ax.set_ylabel('治疗效果')
ax.set_title('药物类型 × 剂量 交互作用图')
ax.legend()
ax.grid(True, alpha=0.3)

# 解读：
# - 如果两条线平行：无交互效应
# - 如果两条线不平行：存在交互效应
# 本例中药物A随剂量上升效果显著增强，药物B变化平缓 → 存在交互效应

plt.show()
```

## Python代码示例

### 使用 statsmodels ols + anova_lm

```python
import pandas as pd
import numpy as np
import statsmodels.api as sm
from statsmodels.formula.api import ols

# 两因素方差分析
# 公式: 因变量 ~ C(因素1) * C(因素2) 表示主效应+交互效应
# 或者: 因变量 ~ C(因素1) + C(因素2) + C(因素1):C(因素2) 完整写法

model = ols('effect ~ C(drug) * C(dose)', data=df).fit()

# Type II ANOVA表
anova_result = sm.stats.anova_lm(model, typ=2)
print("两因素ANOVA结果 (Type II):")
print(anova_result)
print()

# 模型摘要
print(model.summary())
```

### 读取ANOVA表中的关键信息

从ANOVA表中关注三个F检验：

1. **C(drug)**：药物类型的主效应（$p$ 值判断药物间是否有差异）
2. **C(dose)**：剂量的主效应（$p$ 值判断剂量间是否有差异）
3. **C(drug):C(dose)**：药物与剂量的交互效应（$p$ 值判断交互是否显著）

### 交互显著时的策略

```python
# 当交互效应显著时，不能简单解读主效应
# 需要做简单效应分析 (Simple Effects Analysis)

def simple_effects_analysis(df, dv, factor, moderator):
    """简单效应分析：在调节变量的每个水平上，考察因素对因变量的影响"""
    print("简单效应分析:")
    levels = df[moderator].unique()
    for level in levels:
        subset = df[df[moderator] == level]
        model = ols(f'{dv} ~ C({factor})', data=subset).fit()
        anova = sm.stats.anova_lm(model, typ=2)
        f_val = anova.loc[f'C({factor})', 'F']
        p_val = anova.loc[f'C({factor})', 'PR(>F)']
        print(f"  在 {moderator}={level} 水平下:")
        print(f"    {factor}的效应: F = {f_val:.3f}, p = {p_val:.4f}")

# 药物对效应的影响，分别在低/中/高剂量下考察
simple_effects_analysis(df, 'effect', 'drug', 'dose')

# 剂量对效应的影响，分别在药物A和药物B下考察
simple_effects_analysis(df, 'effect', 'dose', 'drug')
```

## 效应量：Partial Eta-squared ($\eta_p^2$)

$$
\eta_p^2 = \frac{SS_{\text{效应}}}{SS_{\text{效应}} + SS_{\text{误差}}}
$$

```python
# 手动计算偏eta-squared
def partial_eta_sq(anova_result, effect):
    """计算偏eta-squared"""
    ss_effect = anova_result.loc[effect, 'sum_sq']
    ss_error = anova_result.loc['Residual', 'sum_sq']
    return ss_effect / (ss_effect + ss_error)

print("\n偏eta-squared:")
for effect in ['C(drug)', 'C(dose)', 'C(drug):C(dose)']:
    if effect in anova_result.index:
        eta = partial_eta_sq(anova_result, effect)
        print(f"  {effect}: η²_p = {eta:.3f}")
```

## 交互效应解读原则

### 交互显著时的解读顺序

```
交互效应显著
    ├── 解读交互效应是首要任务
    ├── 主效应可能具有误导性
    └── 进行简单效应分析
```

### 三原则

1. **交互效应优先**：如果交互显著，首先关注交互效应
2. **主效应谨慎解读**：交互显著时，主效应可能不能反映真实情况
3. **简单效应分析**：在交互显著时，通过简单效应分析揭示交互的本质

### 三因素及以上设计

对于三因素设计（$A \times B \times C$），变异分解为：

$$
SS_{\text{总}} = SS_A + SS_B + SS_C + SS_{A \times B} + SS_{A \times C} + SS_{B \times C} + SS_{A \times B \times C} + SS_{\text{误差}}
$$

## 结果报告模板

### 中文报告

> 采用两因素方差分析（$2 \times 3$ 设计）考察药物类型和剂量对治疗效果的影响。结果显示：药物主效应显著（$F(1, 54) = 32.15, p < 0.001, \eta_p^2 = 0.373$），剂量主效应显著（$F(2, 54) = 28.44, p < 0.001, \eta_p^2 = 0.513$），且药物与剂量的交互效应显著（$F(2, 54) = 12.67, p < 0.001, \eta_p^2 = 0.319$）。简单效应分析表明，药物A的效果随剂量增加显著提升，而药物B的效果受剂量影响较小。在低剂量水平下，药物A与药物B的效果差异不大；但在高剂量水平下，药物A显著优于药物B（$p < 0.001$）。

---

## 关键点总结

1. **交互效应是多因素ANOVA的核心价值**，能揭示单因素分析无法发现的关系
2. 交互作用图中，**线条不平行**提示可能存在交互效应
3. **交互显著时，主效应不能直接解读**，需做简单效应分析
4. 报告时记录交互效应的 $F$ 值、$p$ 值和 $\eta_p^2$
