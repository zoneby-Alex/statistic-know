# 卡方检验原理

## 基本思想

卡方检验（Chi-square test）由 Karl Pearson 于 1900 年提出，用于检验分类变量的**实际观测频数**与**理论期望频数**之间是否存在显著差异。

核心思想是：如果分类变量之间相互独立（或分布符合某种假设），则观察频数与期望频数的差异应仅由随机抽样误差引起；若差异过大，则拒绝原假设。

卡方统计量定义为：

$$
\chi^2 = \sum_{i=1}^{k} \frac{(O_i - E_i)^2}{E_i}
$$

其中 $O_i$ 为观察频数，$E_i$ 为期望频数。该统计量服从自由度为 $(r-1)(c-1)$ 的卡方分布。

## 拟合优度检验 vs 独立性检验

| 类型 | 用途 | 自由度 |
|------|------|--------|
| **拟合优度检验** | 检验单组数据是否符合某种理论分布（如 1:1、1:2:1） | $k-1$（$k$ 为类别数） |
| **独立性检验** | 检验两个分类变量是否相互独立（R×C 表） | $(r-1)(c-1)$ |

**拟合优度检验示例**：检验某地血型分布是否符合 "A:30%, B:25%, O:35%, AB:10%" 的理论比例。

**独立性检验示例**：检验治疗方案（药物/安慰剂）与疗效（有效/无效）是否独立。

## 四格表卡方公式

对于 $2 \times 2$ 四格表：

| | 阳性 | 阴性 | 合计 |
|---|------|------|------|
| 暴露组 | $a$ | $b$ | $a+b$ |
| 非暴露组 | $c$ | $d$ | $c+d$ |
| 合计 | $a+c$ | $b+d$ | $n$ |

计算专用公式（等效于基本公式）：

$$
\chi^2 = \frac{n(ad-bc)^2}{(a+b)(c+d)(a+c)(b+d)}
$$

当 $n \geq 40$ 且所有 $E_i \geq 5$ 时使用；若有 $1 \leq E_i < 5$，需用 **Yates 连续性校正**：

$$
\chi^2 = \sum \frac{(|O_i - E_i| - 0.5)^2}{E_i}
$$

## Python 实现

```python
import numpy as np
from scipy.stats import chi2_contingency

# 四格表数据：治疗组 vs 对照组
#         有效  无效
# 治疗组   45    15
# 对照组   30    30
observed = np.array([[45, 15],
                     [30, 30]])

chi2, p_value, dof, expected = chi2_contingency(observed, correction=True)

print(f"卡方值 = {chi2:.4f}")
print(f"p 值   = {p_value:.4f}")
print(f"自由度 = {dof}")
print("期望频数:")
print(expected)
```

输出示例：

```
卡方值 = 8.2131
p 值   = 0.0042
自由度 = 1
期望频数:
[[37.5 22.5]
 [37.5 22.5]]
```

参数 `correction=True` 启用 Yates 校正（默认仅对 $2\times2$ 表生效）。

## 适用条件

1. **随机样本**：观察值彼此独立
2. **期望频数要求**：
   - 所有 $E_i \geq 5$：直接使用 Pearson 卡方
   - 有 $1 \leq E_i < 5$：使用 Yates 校正或 Fisher 精确检验
   - 有 $E_i < 1$：必须使用 Fisher 精确检验
3. **总样本量**：$n \geq 40$ 较为稳妥
4. **分类互斥**：每个观察值只能属于一个类别
