# 分类数据分析

处理分类（定性）数据的统计方法。

## 内容

| 主题 | 适用场景 | 来源 |
|------|---------|------|
| 卡方检验 | 分类变量分布检验/独立性检验 | 医统第9章 |
| Fisher精确检验 | 小样本列联表 | 医统第9章 / 生统第6章 |
| McNemar检验 | 配对分类数据 | 生统第6章 |
| 列联表关联度量 | 关联强度分析 | 生统第6章 |

## 选择流程

```
结局分类变量
├── 单组分布比较 → 拟合优度卡方
├── 两独立组 → 四格表卡方 / Fisher
├── 两配对组 → McNemar检验
└── R×C表 → 行×列表卡方
```
