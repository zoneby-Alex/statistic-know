
# 集中趋势与离散程度

## 集中趋势（Central Tendency）

集中趋势描述数据向中心值聚集的程度，反映数据分布的中心位置。

### 均值（Mean）

算术均值是最常用的集中趋势指标：

$$\bar{X} = \frac{1}{n}\sum_{i=1}^{n} X_i$$

**特点**：对极端值敏感，适用于对称分布。

### 中位数（Median）

将数据按大小排序后位于中间位置的值：

$$M = 
\begin{cases}
X_{(\frac{n+1}{2})}, & n \text{ 为奇数} \\
\frac{X_{(\frac{n}{2})} + X_{(\frac{n}{2}+1)}}{2}, & n \text{ 为偶数}
\end{cases}$$

**特点**：不受极端值影响，适用于偏态分布。

### 众数（Mode）

数据中出现次数最多的数值。一组数据可能有多个众数或无众数。

**适用场景**：

| 指标 | 适用场景 |
|------|----------|
| 均值 | 对称分布、无极端值 |
| 中位数 | 偏态分布、有极端值 |
| 众数 | 分类数据、离散数据 |

```python
import numpy as np
from scipy import stats

np.random.seed(42)

# 对称分布数据
sym_data = np.random.normal(50, 10, 200)

# 偏态分布数据（含极端值）
skew_data = np.concatenate([
    np.random.normal(50, 10, 190),
    np.array([200, 220, 250, 300, 350, 400, 450, 500, 600, 800])  # 极端值
])

print("===== 对称分布 =====")
print(f"均值 = {np.mean(sym_data):.2f}")
print(f"中位数 = {np.median(sym_data):.2f}")
print(f"众数 = {stats.mode(sym_data, keepdims=True).mode[0]:.2f}")

print("\n===== 偏态分布（含极端值）=====")
print(f"均值 = {np.mean(skew_data):.2f}")
print(f"中位数 = {np.median(skew_data):.2f}")
print("→ 极端值使均值远大于中位数，中位数更能代表"中心"")
```

## 离散程度（Dispersion）

离散程度描述数据的变异大小，反映数据的分散程度。

### 极差（Range）

$$R = X_{\max} - X_{\min}$$

**缺点**：只使用两个极端值，信息利用不充分。

### 方差与标准差

**总体方差**：$\sigma^2 = \frac{1}{N}\sum_{i=1}^{N}(X_i - \mu)^2$

**样本方差**：$S^2 = \frac{1}{n-1}\sum_{i=1}^{n}(X_i - \bar{X})^2$

**样本标准差**：$S = \sqrt{S^2}$

使用 $n-1$（Bessel's correction）而非 $n$ 是为了得到总体方差的无偏估计。

### 四分位数与 IQR

- $Q_1$（第一四分位数）：数据的第 25 百分位数
- $Q_2$（第二四分位数）：中位数，第 50 百分位数
- $Q_3$（第三四分位数）：数据的第 75 百分位数

**四分位距**（Interquartile Range, IQR）：

$$IQR = Q_3 - Q_1$$

IQR 不受极端值影响，常用于偏态分布的变异描述。

```python
# 离散程度计算
print("===== 对称数据 =====")
print(f"极差 = {np.ptp(sym_data):.2f}")
print(f"方差 (ddof=1) = {np.var(sym_data, ddof=1):.2f}")
print(f"标准差 (ddof=1) = {np.std(sym_data, ddof=1):.2f}")
print(f"Q1 = {np.percentile(sym_data, 25):.2f}")
print(f"Q3 = {np.percentile(sym_data, 75):.2f}")
print(f"IQR = {np.percentile(sym_data, 75) - np.percentile(sym_data, 25):.2f}")

print("\n===== 偏态数据 =====")
print(f"极差 = {np.ptp(skew_data):.2f}")
print(f"标准差 (ddof=1) = {np.std(skew_data, ddof=1):.2f}")
print(f"Q1 = {np.percentile(skew_data, 25):.2f}")
print(f"Q3 = {np.percentile(skew_data, 75):.2f}")
print(f"IQR = {np.percentile(skew_data, 75) - np.percentile(skew_data, 25):.2f}")
```

## 变异系数（Coefficient of Variation, CV）

变异系数是标准化后的离散程度指标，用于比较**量纲不同**或**均值差异大**的资料的变异程度：

$$CV = \frac{S}{\bar{X}} \times 100\%$$

```python
# 比较身高 (cm) 和体重 (kg) 的变异程度
heights = np.random.normal(170, 6, 100)   # 均值 170，标准差 6
weights = np.random.normal(65, 10, 100)    # 均值 65，标准差 10

cv_height = np.std(heights, ddof=1) / np.mean(heights) * 100
cv_weight = np.std(weights, ddof=1) / np.mean(weights) * 100

print(f"身高 CV = {cv_height:.2f}%")
print(f"体重 CV = {cv_weight:.2f}%")
print("→ CV 越大，相对变异程度越大")
```

## 综合描述统计

```python
from scipy.stats import describe

data = np.random.normal(50, 10, 200)

# 方法一：scipy 描述统计
nobs, (min_val, max_val), mean, var, skewness, kurtosis = describe(data)
print(f"样本量: {nobs}")
print(f"范围: ({min_val:.2f}, {max_val:.2f})")
print(f"均值: {mean:.2f}")
print(f"方差: {var:.2f}")
print(f"偏度: {skewness:.3f}")
print(f"峰度: {kurtosis:.3f}")

# 方法二：pandas 描述统计（推荐）
import pandas as pd
series = pd.Series(data, name='样本数据')
print("\n" + str(series.describe()))
```
