
# 统计表与统计图

## 统计表的结构与原则

### 基本结构

统计表由**标题**、**标目**（横标目和纵标目）、**线条**和**数字**四部分组成。

```
表 X  标题（简明扼要，概括表格内容）
─────────────────────────────────────
        纵标目1    纵标目2    纵标目3
─────────────────────────────────────
横标目1   数据       数据       数据
横标目2   数据       数据       数据
横标目3   数据       数据       数据
─────────────────────────────────────
注：备注或说明
```

### 制表原则

1. **标题**置于表的上方，居中，简明扼要
2. **标目**分类清晰，纵标目指明指标名称和单位
3. **线条**：通常采用"三线表"（顶线、表头线、底线），可加辅助线
4. **数字**：小数点对齐，同列单位一致，缺失用"—"表示
5. **备注**不应写入表内，用脚注说明

```python
import pandas as pd
import numpy as np

# 创建规范的三线表格式
np.random.seed(42)
groups = ['对照组', '低剂量组', '高剂量组']
n_per_group = 30

data = pd.DataFrame({
    '组别': groups,
    '例数': [n_per_group] * 3,
    '均值': np.random.normal([120, 125, 130], 5, 3).round(1),
    '标准差': np.random.uniform(4, 8, 3).round(2),
    't值': [np.nan, 2.15, 3.87],
    'P值': [np.nan, 0.036, 0.001]
})
data['均值±标准差'] = data['均值'].astype(str) + ' ± ' + data['标准差'].astype(str)
print("三线表示例：治疗前后血压比较")
print("=" * 60)
print(data[['组别', '例数', '均值±标准差', 't值', 'P值']].to_string(index=False))
```

## 常用统计图

### 1. 条图（Bar Chart）

用于展示分类数据的数值大小。横轴为分类变量，纵轴为统计指标。

```python
import matplotlib.pyplot as plt
import numpy as np

plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

categories = ['对照组', '低剂量', '中剂量', '高剂量']
values = [100, 115, 128, 135]
errors = [8, 9, 10, 11]

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 5))

# 单式条图
ax1.bar(categories, values, yerr=errors, capsize=5,
        color=['#4C72B0', '#55A868', '#C44E52', '#8172B2'])
ax1.set_title('不同治疗组的疗效比较')
ax1.set_ylabel('疗效指标')

# 复式条图
x = np.arange(3)
width = 0.35
ax2.bar(x - width/2, [85, 92, 88], width, label='治疗前', color='#4C72B0')
ax2.bar(x + width/2, [100, 115, 128], width, label='治疗后', color='#55A868')
ax2.set_xticks(x)
ax2.set_xticklabels(['对照组', '低剂量', '高剂量'])
ax2.set_title('治疗前后对比')
ax2.legend()
plt.tight_layout()
plt.show()
```

### 2. 线图（Line Chart）

用于展示指标随连续变量（如时间、剂量）变化的趋势。

```python
fig, ax = plt.subplots(figsize=(8, 5))

weeks = np.arange(1, 13)
group_A = 100 + 5 * weeks + np.random.normal(0, 3, 12)
group_B = 100 + 3 * weeks + np.random.normal(0, 3, 12)

ax.plot(weeks, group_A, 'o-', label='治疗组 A', linewidth=2)
ax.plot(weeks, group_B, 's--', label='治疗组 B', linewidth=2)
ax.set_xlabel('治疗周数')
ax.set_ylabel('指标值')
ax.set_title('治疗过程中指标变化趋势')
ax.legend()
ax.grid(True, alpha=0.3)
plt.show()
```

### 3. 散点图（Scatter Plot）

用于展示两个连续变量之间的关系。

```python
fig, ax = plt.subplots(figsize=(7, 6))

age = np.random.uniform(20, 80, 100)
sbp = 110 + 0.5 * age + np.random.normal(0, 8, 100)

ax.scatter(age, sbp, alpha=0.6, color='steelblue', edgecolors='white')
# 添加回归线
z = np.polyfit(age, sbp, 1)
p = np.poly1d(z)
x_line = np.linspace(20, 80, 100)
ax.plot(x_line, p(x_line), 'r--', linewidth=2)
ax.set_xlabel('年龄 (岁)')
ax.set_ylabel('收缩压 (mmHg)')
ax.set_title('年龄与收缩压的关系')
plt.show()
```

### 4. 箱线图（Box Plot）

详见 [频数分布与直方图](频数分布与直方图.md)，此处给出医学论文中分组的箱线图规范示例。

```python
fig, ax = plt.subplots(figsize=(8, 5))

data_groups = [
    np.random.normal(120, 8, 40),
    np.random.normal(125, 9, 40),
    np.random.normal(135, 10, 40)
]

bp = ax.boxplot(data_groups, labels=['对照组', '模型组', '给药组'],
                patch_artist=True, widths=0.5,
                medianprops={'color': 'red', 'linewidth': 2})
colors = ['#4C72B0', '#C44E52', '#55A868']
for patch, color in zip(bp['boxes'], colors):
    patch.set_facecolor(color)
ax.set_ylabel('血压 (mmHg)')
ax.set_title('各组血压水平比较')
ax.grid(True, axis='y', alpha=0.3)
plt.show()
```

## 医学论文中统计图的规范要点

| 要求 | 说明 |
|------|------|
| 标题 | 置于图的下方，居中，包含图序号 |
| 坐标轴 | 标明名称和单位，刻度均匀 |
| 图例 | 清晰区分不同组别或变量 |
| 分辨率 | 一般要求 $\ge 300$ dpi |
| 颜色 | 考虑黑白印刷时用不同线型或符号区分 |
| 比例 | 避免拉长或压扁图形，误导读者 |
| 导出格式 | TIFF 用于期刊投稿，PNG 用于网络展示 |

```python
# 符合期刊规范的图保存
fig.savefig('figure_1.tiff', dpi=300, format='tiff',
            bbox_inches='tight', facecolor='white')
fig.savefig('figure_1.png', dpi=300, format='png',
            bbox_inches='tight', facecolor='white')
```
