
# 频数分布与直方图

## 频数分布表

**频数分布**（frequency distribution）是将原始数据按数值大小分组后，统计各组内数据个数（频数）所形成的表格。它是对数据进行初步整理和探索性分析的基础工具。

### 构建频数分布表的步骤

1. **确定组数** $k$：通常取 $5 \sim 15$ 组，常用经验公式 $k \approx 1 + 3.322\log_{10}(n)$
2. **确定组距** $h = \frac{\max - \min}{k}$，通常取整数值
3. **确定组限**：各组的下限和上限
4. **统计频数**：计数落入各组的观测值个数

```python
import numpy as np
import pandas as pd

# 生成示例数据：100 名成年男性的身高 (cm)
np.random.seed(42)
heights = np.random.normal(170, 6, 100)

# 构建频数分布表
k = int(1 + 3.322 * np.log10(len(heights)))  # Sturges 公式
h = (heights.max() - heights.min()) / k
h = round(h)  # 取整

bins = np.arange(heights.min(), heights.max() + h, h)
labels = [f"[{bins[i]:.0f}, {bins[i+1]:.0f})" for i in range(len(bins)-1)]
freq_table = pd.cut(heights, bins=bins, labels=labels, right=False)

# 输出频数表
freq_df = pd.DataFrame({
    '组段': labels,
    '频数': pd.value_counts(freq_table).sort_index().values
})
freq_df['频率(%)'] = freq_df['频数'] / len(heights) * 100
freq_df['累积频数'] = freq_df['频数'].cumsum()
freq_df['累积频率(%)'] = freq_df['频率(%)'].cumsum()
print("频数分布表：")
print(freq_df.to_string(index=False))
```

## 直方图（Histogram）

直方图是展示定量数据分布形态最常用的图形工具。横轴为变量取值（分组），纵轴为频数或频率密度。

```python
import matplotlib.pyplot as plt
import seaborn as sns

# 设置中文字体
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

fig, axes = plt.subplots(1, 3, figsize=(15, 4))

# 1. 基本直方图
axes[0].hist(heights, bins=12, edgecolor='white', color='steelblue')
axes[0].set_title('频数直方图')
axes[0].set_xlabel('身高 (cm)')
axes[0].set_ylabel('频数')

# 2. 频率密度直方图 + 核密度曲线
axes[1].hist(heights, bins=12, density=True, edgecolor='white',
             alpha=0.7, color='steelblue')
sns.kdeplot(heights, ax=axes[1], color='darkred', linewidth=2)
axes[1].set_title('密度直方图 + KDE')
axes[1].set_xlabel('身高 (cm)')

# 3. 分组直方图（用不同颜色表示组别）
group1 = np.random.normal(165, 5, 80)
group2 = np.random.normal(175, 5, 80)
axes[2].hist(group1, bins=10, alpha=0.6, label='Group A')
axes[2].hist(group2, bins=10, alpha=0.6, label='Group B')
axes[2].set_title('分组直方图对比')
axes[2].legend()

plt.tight_layout()
plt.show()
```

## 箱线图（Box Plot）

箱线图通过五个统计量展示数据的分布特征：最小值、$Q_1$（第一四分位数）、中位数、$Q_3$（第三四分位数）、最大值，并标识异常值。

**箱线图的构成**：

- **箱体**：$Q_1 \to Q_3$（即 IQR 区间）
- **箱中横线**：中位数
- **须**（whisker）：$Q_1 - 1.5 \times IQR$ 到 $Q_3 + 1.5 \times IQR$
- **离群点**：超出须范围的点

```python
# 箱线图示例
fig, axes = plt.subplots(1, 2, figsize=(12, 5))

# 单一箱线图
axes[0].boxplot(heights, widths=0.5, patch_artist=True,
                boxprops={'facecolor': 'lightblue'},
                medianprops={'color': 'red', 'linewidth': 2})
axes[0].set_title('身高箱线图')
axes[0].set_ylabel('身高 (cm)')
axes[0].set_xticklabels(['样本'])

# 多组箱线图
data_groups = [
    np.random.normal(160, 5, 50),
    np.random.normal(170, 6, 50),
    np.random.normal(180, 7, 50)
]
bp = axes[1].boxplot(data_groups, labels=['Group 1', 'Group 2', 'Group 3'],
                      patch_artist=True)
colors = ['lightcoral', 'lightblue', 'lightgreen']
for patch, color in zip(bp['boxes'], colors):
    patch.set_facecolor(color)
axes[1].set_title('多组箱线图对比')
axes[1].set_ylabel('值')

plt.tight_layout()
plt.show()
```

## 直方图与箱线图的配合使用

两种图形各有优势，建议配合使用：

| 图形 | 优势 | 劣势 |
|------|------|------|
| 直方图 | 展示分布形态（偏态、双峰等） | 对组距敏感 |
| 箱线图 | 展示离散程度、异常值 | 无法展示多峰分布 |

```python
# 联合展示
fig, (ax_top, ax_bottom) = plt.subplots(2, 1, figsize=(10, 6),
                                          gridspec_kw={'height_ratios': [3, 1]})

ax_top.hist(heights, bins=15, edgecolor='white', color='steelblue')
ax_top.set_ylabel('频数')

ax_bottom.boxplot(heights, vert=False, widths=0.5, patch_artist=True,
                  boxprops={'facecolor': 'lightblue'})
ax_bottom.set_xlabel('身高 (cm)')

plt.tight_layout()
plt.show()
```
