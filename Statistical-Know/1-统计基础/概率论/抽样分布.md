
# 抽样分布

## 基本概念

**抽样分布**（sampling distribution）是指统计量的概率分布。当我们从总体中重复抽取样本时，每次计算同一个统计量（如样本均值 $\bar{X}$），这些统计量的分布即为抽样分布。

理解抽样分布是统计推断的核心基础。

---

## 中心极限定理（Central Limit Theorem, CLT）

**中心极限定理**是统计学中最重要的定理之一：

> 设 $X_1, X_2, \dots, X_n$ 是来自均值为 $\mu$、方差为 $\sigma^2$ 的任意总体的独立同分布样本，当样本量 $n$ 充分大时，样本均值 $\bar{X} = \frac{1}{n}\sum_{i=1}^{n} X_i$ 的抽样分布近似服从正态分布：
>
> $$\bar{X} \;\overset{\cdot}{\sim}\; N\!\left(\mu,\; \frac{\sigma^2}{n}\right)$$

**重要意义**：
- 无论总体分布形态如何，样本均值的抽样分布都趋向正态
- 抽样分布的方差随 $n$ 增大而减小（估计精度提高）
- 通常 $n \ge 30$ 可认为"充分大"

### Python 模拟 CLT

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import expon, norm

# 从指数分布（偏态分布）中抽样
population = expon(scale=2)  # 均值=2，方差=4
mu, sigma = 2, 2

fig, axes = plt.subplots(2, 3, figsize=(14, 8))
sample_sizes = [1, 5, 10, 20, 50, 100]

for ax, n in zip(axes.flatten(), sample_sizes):
    # 重复抽样 10000 次，每次计算样本均值
    sample_means = [np.mean(population.rvs(n)) for _ in range(10000)]
    ax.hist(sample_means, bins=40, density=True, alpha=0.7, edgecolor='white')
    
    # 叠加正态分布曲线（CLT 理论）
    x = np.linspace(min(sample_means), max(sample_means), 200)
    ax.plot(x, norm.pdf(x, mu, sigma / np.sqrt(n)),
            'r-', linewidth=2, label='N(μ, σ²/n)')
    ax.set_title(f'n = {n}', fontsize=12)
    ax.legend()

plt.tight_layout()
plt.show()
```

> 当 $n=1$ 时，抽样分布就是总体分布本身（指数分布，严重偏态）；随着 $n$ 增大，分布逐渐趋向正态。

## 三大抽样分布

### 1. $\chi^2$ 分布（卡方分布）

若 $Z_1, Z_2, \dots, Z_\nu \overset{\text{iid}}{\sim} N(0,1)$，则

$$\chi^2 = \sum_{i=1}^{\nu} Z_i^2 \sim \chi^2(\nu)$$

$\nu$ 为**自由度**。从正态总体 $N(\mu, \sigma^2)$ 中抽样时：

$$\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$$

其中 $S^2 = \frac{1}{n-1}\sum_{i=1}^{n}(X_i - \bar{X})^2$ 为样本方差。

### 2. t 分布（Student's t 分布）

若 $Z \sim N(0,1)$ 与 $Y \sim \chi^2(\nu)$ 独立，则：

$$t = \frac{Z}{\sqrt{Y/\nu}} \sim t(\nu)$$

**由来**：当总体方差 $\sigma^2$ 未知时，用样本标准差 $S$ 代替：

$$t = \frac{\bar{X} - \mu}{S/\sqrt{n}} \sim t(n-1)$$

t 分布比标准正态分布尾部更厚，当 $\nu \to \infty$ 时趋近于 $N(0,1)$。

### 3. F 分布

若 $U \sim \chi^2(\nu_1)$ 与 $V \sim \chi^2(\nu_2)$ 独立，则：

$$F = \frac{U/\nu_1}{V/\nu_2} \sim F(\nu_1, \nu_2)$$

**由来**：比较两个正态总体的方差时：

$$F = \frac{S_1^2/\sigma_1^2}{S_2^2/\sigma_2^2} \sim F(n_1-1, n_2-1)$$

在 ANOVA（方差分析）中，F 分布用于检验多个组均值是否相等。

---

## 三大分布之间的关系

```python
from scipy.stats import t, norm, chi2, f
import numpy as np

# 1. t 分布 → 标准正态分布（自由度增大时）
df_list = [2, 5, 30]
for df_val in df_list:
    t_95 = t.ppf(0.975, df_val)
    z_95_v = norm.ppf(0.975)
    print(f"t({df_val}) 临界值 = {t_95:.4f}, N(0,1) 临界值 = {z_95_v:.4f}")

# 2. t^2 = F(1, ν)
df_t = 10
t_val = t.ppf(0.975, df_t)
f_val = f.ppf(0.95, 1, df_t)
print(f"\nt({df_t})² = {t_val**2:.4f}")
print(f"F(1, {df_t}) = {f_val:.4f}")

# 3. χ² 分布的可加性
df1, df2 = 3, 5
# χ²(3) + χ²(5) → χ²(8)
sample1 = chi2.rvs(df1, size=10000)
sample2 = chi2.rvs(df2, size=10000)
sample_sum = sample1 + sample2
print(f"\nχ²(3)+χ²(5) 样本均值 = {sample_sum.mean():.3f}, 理论值 = {df1+df2}")
```

---

## 常用结论总结

| 统计量 | 分布 | 自由度 |
|--------|------|--------|
| $\dfrac{\bar{X} - \mu}{\sigma/\sqrt{n}}$ | $N(0,1)$ | — |
| $\dfrac{\bar{X} - \mu}{S/\sqrt{n}}$ | $t(n-1)$ | $n-1$ |
| $\dfrac{(n-1)S^2}{\sigma^2}$ | $\chi^2(n-1)$ | $n-1$ |
| $\dfrac{S_1^2/\sigma_1^2}{S_2^2/\sigma_2^2}$ | $F(n_1-1, n_2-1)$ | $n_1-1, n_2-1$ |
