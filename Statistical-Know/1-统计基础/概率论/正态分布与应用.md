
# 正态分布与应用

## 正态分布的特性

正态分布（Gaussian 分布）是统计学中最重要且应用最广泛的连续概率分布。若随机变量 $X$ 服从均值为 $\mu$、方差为 $\sigma^2$ 的正态分布，记作 $X \sim N(\mu, \sigma^2)$。

**概率密度函数**：

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} \exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right),\quad x \in \mathbb{R}$$

### 68-95-99.7 规则（经验规则）

对于正态分布，数据落在均值附近指定范围内的概率是固定的：

- 约 **68%** 的数据落在 $(\mu - \sigma,\; \mu + \sigma)$ 内
- 约 **95%** 的数据落在 $(\mu - 1.96\sigma,\; \mu + 1.96\sigma)$ 内
- 约 **99.7%** 的数据落在 $(\mu - 3\sigma,\; \mu + 3\sigma)$ 内

```python
from scipy.stats import norm
import numpy as np

mu, sigma = 170, 6

within_1sd = norm.cdf(mu + sigma, mu, sigma) - norm.cdf(mu - sigma, mu, sigma)
within_2sd = norm.cdf(mu + 1.96*sigma, mu, sigma) - norm.cdf(mu - 1.96*sigma, mu, sigma)
within_3sd = norm.cdf(mu + 3*sigma, mu, sigma) - norm.cdf(mu - 3*sigma, mu, sigma)

print(f"μ±σ 范围内比例: {within_1sd:.3f} (理论68.3%)")
print(f"μ±1.96σ 范围内比例: {within_2sd:.3f} (理论95.0%)")
print(f"μ±3σ 范围内比例: {within_3sd:.3f} (理论99.7%)")
```

## 标准正态分布与标准化

### Z 标准化（Z-score）

对于任意 $X \sim N(\mu, \sigma^2)$，进行线性变换：

$$Z = \frac{X - \mu}{\sigma} \sim N(0, 1)$$

标准正态分布 $N(0,1)$ 的累积分布函数常用 $\Phi(z)$ 表示。

```python
from scipy.stats import norm

# 给定身高 175cm，总体 μ=170, σ=6
x, mu, sigma = 175, 170, 6
z = (x - mu) / sigma
print(f"Z-score = {z:.4f}")

# 计算超过 175cm 的概率
prob = 1 - norm.cdf(z)
print(f"P(X > 175) = {prob:.4f}")

# 查找百分位数：找到使得 P(X < x) = 0.95 的 x 值
p95 = norm.ppf(0.95, mu, sigma)
print(f"第95百分位数 = {p95:.2f}cm")
```

## 正态性检验

许多统计方法（如 t 检验、ANOVA）要求数据近似正态分布。常用检验方法：

### 1. Shapiro-Wilk 检验

最常用的正态性检验之一，适用于小样本（$n < 5000$）。

$$W = \frac{\left(\sum_{i=1}^{n} a_i X_{(i)}\right)^2}{\sum_{i=1}^{n} (X_i - \bar{X})^2}$$

$H_0$：数据服从正态分布。若 $p < 0.05$，拒绝 $H_0$，认为数据不服从正态分布。

```python
from scipy.stats import shapiro, probplot
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)
# 生成正态数据
normal_data = np.random.normal(170, 6, 100)
# 生成偏态数据
skewed_data = np.random.exponential(10, 100)

stat_n, p_n = shapiro(normal_data)
stat_s, p_s = shapiro(skewed_data)

print(f"正态数据: W={stat_n:.4f}, p={p_n:.4f} -> {'正态' if p_n > 0.05 else '非正态'}")
print(f"偏态数据: W={stat_s:.4f}, p={p_s:.4f} -> {'正态' if p_s > 0.05 else '非正态'}")
```

### 2. Q-Q 图（Quantile-Quantile Plot）

将样本分位数与理论正态分位数作图。若数据服从正态分布，点应近似落在 $y=x$ 直线上。

```python
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 5))

# 正态数据 Q-Q 图
probplot(normal_data, dist="norm", plot=ax1)
ax1.set_title("正态数据 Q-Q 图")

# 偏态数据 Q-Q 图
probplot(skewed_data, dist="norm", plot=ax2)
ax2.set_title("偏态数据 Q-Q 图")

plt.tight_layout()
plt.show()
```

## 医学参考值范围计算

**医学参考值范围**（reference range）是指绝大多数"正常人"的某项指标测量值所在的范围。

### 正态分布法（适用于正态或近似正态数据）

$$95\%\text{参考值范围}: \bar{X} \pm 1.96 \times S$$

- 双侧：$\bar{X} \pm Z_{\alpha/2} \times S$
- 单侧上限：$\bar{X} + Z_{\alpha} \times S$（如血铅）
- 单侧下限：$\bar{X} - Z_{\alpha} \times S$（如肺活量）

### 百分位数法（适用于偏态数据）

- 双侧 95% 范围：$P_{2.5} \sim P_{97.5}$
- 单侧 95% 上限：$P_{95}$
- 单侧 95% 下限：$P_5$

```python
from scipy.stats import norm
import numpy as np

# 示例数据：100 名健康成人血清胆固醇 (mmol/L)
np.random.seed(123)
data = np.random.normal(4.5, 0.8, 100)

# 正态分布法
mean = np.mean(data)
std = np.std(data, ddof=1)
lower_norm = mean - 1.96 * std
upper_norm = mean + 1.96 * std
print(f"正态分布法 95% 参考值范围: ({lower_norm:.2f}, {upper_norm:.2f})")

# 百分位数法
lower_pct = np.percentile(data, 2.5)
upper_pct = np.percentile(data, 97.5)
print(f"百分位数法 95% 参考值范围: ({lower_pct:.2f}, {upper_pct:.2f})")

# 比较两种方法
print(f"\n两种方法差异: 下限差={abs(lower_norm-lower_pct):.3f}, 上限差={abs(upper_norm-upper_pct):.3f}")

# 医学上的选择建议
print("\n选择建议:")
print("- 正态分布 → 正态分布法（更稳定）")
print("- 偏态分布 → 百分位数法（无需正态假设）")
```
