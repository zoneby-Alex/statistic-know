
# 标准化法

## 为什么要标准化

在比较两个人群的总体率（如死亡率、发病率）时，若两组人群的**内部构成**（如年龄、性别）不同，直接比较总率会产生偏倚。**标准化法**（standardization）通过选择统一的标准结构，消除内部构成差异的影响，使总率可比。

**典型场景**：
- 比较不同地区的死亡率（人口年龄结构不同）
- 比较不同医院的治愈率（患者病情构成不同）
- 不同年代的率比较（人口老龄化趋势）

---

## 直接标准化法

**适用条件**：已知各组的**年龄别（分层）率**。

**计算步骤**：
1. 选择**标准人口**（如全国人口、世界标准人口）
2. 用标准人口各层人口数 $N_i$ 乘以各组的年龄别率 $p_i$
3. 求和后除以标准总人口

$$\text{标准化率} = \frac{\sum N_i p_i}{\sum N_i} \times K$$

其中 $N_i$ 为标准人口第 $i$ 层的人口数，$p_i$ 为被标准化组第 $i$ 层的率。

```python
import pandas as pd
import numpy as np

# 直接标准化：比较 A、B 两地的总死亡率
# 已知标准人口（2020年全国人口）和各年龄别死亡率
std_pop = pd.DataFrame({
    '年龄组': ['0-', '15-', '30-', '45-', '60-', '70+'],
    '标准人口': [15000, 20000, 25000, 20000, 12000, 8000]  # 万人
})

# A 地各年龄别死亡率 (1/10万)
A_rates = np.array([120, 80, 150, 400, 1200, 3500])
# B 地各年龄别死亡率 (1/10万)
B_rates = np.array([100, 70, 130, 380, 1150, 3400])
# A 地人口构成（与标准不同）
A_pop = np.array([18000, 22000, 20000, 15000, 10000, 5000])

std_total = std_pop['标准人口'].sum()

# 直接标准化
A_expected = (std_pop['标准人口'] * A_rates).sum() / std_total
B_expected = (std_pop['标准人口'] * B_rates).sum() / std_total

# 未标准化的粗率（用各自人口）
A_crude = (A_pop * A_rates).sum() / A_pop.sum()
B_crude = (A_pop * B_rates).sum() / A_pop.sum()  # 假设 B 地人口与 A 地相同便于比较

print("直接标准化结果")
print("=" * 50)
print(f"A 地标准化死亡率: {A_expected:.1f}/10万")
print(f"B 地标准化死亡率: {B_expected:.1f}/10万")
print(f"\nA 地各年龄别死亡率均高于 B 地")
print(f"但若 A 地人口整体偏年轻 → 未标准化时粗率可能偏低")
print(f"标准化后：A 地 ({A_expected:.1f}) > B 地 ({B_expected:.1f})")
print("→ 标准化消除了人口年龄构成差异的影响")
```

## 间接标准化法

**适用条件**：已知**标准人群的年龄别率**和**被比较组的总死亡人数及年龄别人口数**。

**计算步骤**：
1. 用标准人群的年龄别率 $p_i$ 乘被比较组各层人口数 $n_i$，得到**期望死亡数**
2. 计算**标准化死亡比**（Standardized Mortality Ratio, SMR）：

$$SMR = \frac{\text{实际死亡数}}{\text{期望死亡数}} = \frac{O}{\sum n_i p_i}$$

3. 间接标准化率 $= \text{标准人群总率} \times SMR$

```python
# 间接标准化：计算某工厂工人的 SMR
# 标准人群（全国人口）的年龄别死亡率
std_rates = pd.DataFrame({
    '年龄组': ['20-', '30-', '40-', '50-', '60-'],
    '标准死亡率(1/10万)': [50, 80, 200, 500, 1500]
})

# 某工厂工人的人口构成和实际死亡数
factory_pop = np.array([3000, 2500, 2000, 1000, 500])  # 各年龄组人数
observed_deaths = 12  # 实际观察到的总死亡数

# 计算期望死亡数
expected_deaths = (factory_pop * std_rates['标准死亡率(1/10万)'] / 100000).sum()
smr = observed_deaths / expected_deaths

# 标准人群总死亡率
std_total_rate = ((std_rates['标准死亡率(1/10万)']).sum() / len(std_rates)) / 100000

# 工厂间接标准化死亡率
factory_adj_rate = std_total_rate * smr * 100000  # 转为 /10万

print("间接标准化结果")
print("=" * 50)
print(f"期望死亡数（基于标准人群）: {expected_deaths:.2f} 人")
print(f"实际死亡数: {observed_deaths} 人")
print(f"SMR (标准化死亡比) = {observed_deaths} / {expected_deaths:.2f} = {smr:.3f}")
print(f"\nSMR 解读:")
if smr > 1:
    print(f"  SMR = {smr:.3f} > 1 → 该工厂死亡风险高于一般人群")
elif smr < 1:
    print(f"  SMR = {smr:.3f} < 1 → 该工厂死亡风险低于一般人群")
else:
    print(f"  SMR = {smr:.3f} = 1 → 该工厂死亡风险与一般人群相当")
print(f"\n工厂间接标准化死亡率: {factory_adj_rate:.1f}/10万")
```

## 直接标准化 vs 间接标准化

| 比较项 | 直接标准化 | 间接标准化 |
|--------|-----------|-----------|
| 所需数据 | 各组的年龄别率 | 各组的年龄别人口数和总死亡数 |
| 所需标准 | 标准人口构成 | 标准人群的年龄别率 |
| 适用场景 | 各组年龄别率已知且稳定 | 各组年龄别率不稳定或未知 |
| 稳定性 | 各层例数较小时率不稳定 | 相对更稳定 |
| 常见应用 | 世界人口标准化率 | SMR 职业病流行病学研究 |

### 重要注意事项

1. **标准化率的可比性**：标准化率是相对值而非真实值，仅在统一标准下具有可比性
2. **标准的选择**：使用不同标准会导致不同的标准化率
3. **精度的损失**：标准化过程会掩盖组内差异
4. **显著性检验**：标准化率之间也需要进行统计检验

## 适用场景总结

标准化法在生物医学领域中广泛应用：

- **肿瘤登记**：世界标准人口标准化发病率（ASIR）
- **医院质量评估**：标准化死亡率（HSMR）
- **职业病研究**：SMR 评估职业暴露风险
- **国际比较**：统一标准后的跨国死亡率比较
