
# 常用统计指标

## 疾病频率指标

### 发病率（Incidence Rate）

**发病率**表示在一定时期内，特定人群中**新发**某病病例的频率。

$$\text{发病率} = \frac{\text{某时期内新发病例数}}{\text{同期暴露人口数}} \times K$$

- 反映疾病的**发生风险**
- 多用于急性病、传染病的监测
- 比例基数 $K$ 根据疾病频率选取（$1000\%$、$100000/10\text{万}$ 等）

### 患病率（Prevalence Rate）

**患病率**表示在特定时间点（或时期），某人群中**现存**某病病例（新旧病例合计）所占的比例。

$$\text{患病率} = \frac{\text{特定时点现存病例数}}{\text{同期调查人数}} \times K$$

- 反映疾病的**负担**
- 多用于慢性病（如高血压、糖尿病）
- **患病率 $\approx$ 发病率 $\times$ 病程**

```python
import numpy as np

# 发病率与患病率的关系示例
# 假设某地区
population = 100000  # 总人口
new_cases_2023 = 500  # 2023年新发病例数
existing_cases = 2500  # 2023年底现有病例数

incidence = new_cases_2023 / population * 100000
prevalence = existing_cases / population * 100000

print(f"发病率 = {incidence:.1f}/10万")
print(f"患病率 = {prevalence:.1f}/10万")
print(f"患病率/发病率 ≈ {prevalence/incidence:.1f} ≈ 平均病程(年)")
```

### 死亡率（Mortality Rate）

$$\text{死亡率} = \frac{\text{某时期内死亡人数}}{\text{同期平均人口数}} \times K$$

### 生存率（Survival Rate）

常用 **Kaplan-Meier 法** 估计（详见生存分析章节）。

## 诊断试验评价指标

以 $2 \times 2$ 列联表为基础：

| 诊断结果 | 金标准阳性 | 金标准阴性 | 合计 |
|---------|-----------|-----------|------|
| 阳性 | $a$ （真阳性） | $b$ （假阳性） | $a+b$ |
| 阴性 | $c$ （假阴性） | $d$ （真阴性） | $c+d$ |
| 合计 | $a+c$ | $b+d$ | $N$ |

### 敏感度（Sensitivity, Se）

$$\text{Se} = \frac{a}{a+c} \times 100\%$$

表示患者中被正确诊断为阳性的比例。敏感度越高，**漏诊率**越低。

### 特异度（Specificity, Sp）

$$\text{Sp} = \frac{d}{b+d} \times 100\%$$

表示非患者中被正确诊断为阴性的比例。特异度越高，**误诊率**越低。

### 阳性预测值（Positive Predictive Value, PPV）

$$\text{PPV} = \frac{a}{a+b} \times 100\%$$

表示诊断阳性者中真正患病的比例。PPV 受**患病率**影响。

### 阴性预测值（Negative Predictive Value, NPV）

$$\text{NPV} = \frac{d}{c+d} \times 100\%$$

表示诊断阴性者中真正未患病的比例。

```python
# 诊断试验评价
def diagnostic_test(tp, fp, fn, tn):
    """
    计算诊断试验评价指标

    参数：
    tp: 真阳性, fp: 假阳性, fn: 假阴性, tn: 真阴性
    """
    total = tp + fp + fn + tn
    sensitivity = tp / (tp + fn) * 100
    specificity = tn / (tn + fp) * 100
    ppv = tp / (tp + fp) * 100
    npv = tn / (tn + fn) * 100
    accuracy = (tp + tn) / total * 100
    prevalence = (tp + fn) / total * 100

    return {
        '敏感度(Se)': f"{sensitivity:.1f}%",
        '特异度(Sp)': f"{specificity:.1f}%",
        '阳性预测值(PPV)': f"{ppv:.1f}%",
        '阴性预测值(NPV)': f"{npv:.1f}%",
        '准确度': f"{accuracy:.1f}%",
        '患病率': f"{prevalence:.1f}%",
        'Youden指数': f"{sensitivity + specificity - 100:.1f}%"
    }

# 例：某肿瘤标志物检测评价
result = diagnostic_test(tp=85, fp=20, fn=15, tn=180)
print("肿瘤标志物诊断试验评价")
print("=" * 40)
for k, v in result.items():
    print(f"{k}: {v}")

# 患病率对 PPV 的影响
print("\n患病率对 PPV 的影响")
print("=" * 50)
prevalences = [0.5, 1, 5, 10, 20]
se, sp = 0.90, 0.95  # 固定敏感度和特异度
for prev in prevalences:
    N = 10000
    diseased = int(N * prev / 100)
    healthy = N - diseased
    tp = int(diseased * se)
    fn = diseased - tp
    tn = int(healthy * sp)
    fp = healthy - tn
    ppv = tp / (tp + fp) * 100
    print(f"患病率={prev}% → PPV={ppv:.1f}%")
print("→ 患病率越低，PPV 越低（即使敏感度和特异度都很高）")
```

## RR 与 OR

### 相对危险度（Relative Risk, RR）

$$\text{RR} = \frac{\text{暴露组的发病率}}{\text{非暴露组的发病率}} = \frac{a/(a+b)}{c/(c+d)}$$

- 适用于**队列研究**
- $\text{RR} > 1$：暴露为危险因素
- $\text{RR} < 1$：暴露为保护因素
- $\text{RR} = 1$：暴露与疾病无关

### 比值比（Odds Ratio, OR）

$$OR = \frac{a/c}{b/d} = \frac{ad}{bc}$$

- 适用于**病例对照研究**（无法直接计算发病率时）
- 对罕见病，$OR \approx RR$

```python
# RR 和 OR 的计算
def calc_effect_measures(a, b, c, d):
    """
    a=暴露+发病, b=暴露+未发病
    c=非暴露+发病, d=非暴露+未发病
    """
    # RR
    risk_exposed = a / (a + b)
    risk_unexposed = c / (c + d)
    rr = risk_exposed / risk_unexposed

    # OR
    or_val = (a * d) / (b * c)

    # 95% 可信区间（Woolf 法）
    se_log_rr = np.sqrt(1/a - 1/(a+b) + 1/c - 1/(c+d))
    se_log_or = np.sqrt(1/a + 1/b + 1/c + 1/d)

    rr_ci = (np.exp(np.log(rr) - 1.96*se_log_rr),
             np.exp(np.log(rr) + 1.96*se_log_rr))
    or_ci = (np.exp(np.log(or_val) - 1.96*se_log_or),
             np.exp(np.log(or_val) + 1.96*se_log_or))

    return {
        'RR': f"{rr:.3f} ({rr_ci[0]:.3f}, {rr_ci[1]:.3f})",
        'OR': f"{or_val:.3f} ({or_ci[0]:.3f}, {or_ci[1]:.3f})",
        '暴露组发病率': f"{risk_exposed:.1%}",
        '非暴露组发病率': f"{risk_unexposed:.1%}"
    }

# 队列研究示例：吸烟与肺癌
print("队列研究：吸烟与肺癌")
result = calc_effect_measures(a=60, b=3940, c=10, d=3990)
for k, v in result.items():
    print(f"{k}: {v}")

# 病例对照研究示例
print("\n病例对照研究：吸烟与肺癌")
result_cc = calc_effect_measures(a=60, b=40, c=10, d=90)
for k, v in result_cc.items():
    print(f"{k}: {v}")

# RR 与 OR 的关系
print("\n发病率对 RR 与 OR 差异的影响")
# 罕见病时 OR ≈ RR
rare = calc_effect_measures(a=20, b=9980, c=5, d=9995)
print(f"罕见病: RR={rare['RR']}, OR={rare['OR']}")
# 常见病时 OR ≠ RR
common = calc_effect_measures(a=400, b=600, c=200, d=800)
print(f"常见病: RR={common['RR']}, OR={common['OR']}")
```

## 指标选择总结

| 研究设计 | 推荐指标 | 原因 |
|---------|---------|------|
| 队列研究 | **RR** | 可直接计算发病率 |
| 病例对照研究 | **OR** | 无法计算发病率 |
| 横断面研究 | **患病率** | 反映疾病负担 |
| 诊断试验 | **Se, Sp, PPV, NPV** | 评价诊断准确性 |
| 生态学研究 | **相关系数** | 群体水平分析 |
