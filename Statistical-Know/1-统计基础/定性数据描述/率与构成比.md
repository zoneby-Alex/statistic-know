
# 率与构成比

## 相对数的基本概念

在医学研究中，定性数据（分类数据）常通过计算**相对数**来进行比较和分析。常用的相对数包括率、构成比和相对比。

### 1. 率（Rate）

**率**是指在一定时期内，某现象实际发生的次数与可能发生的总次数之比，反映事件发生的**频率或强度**。

$$\text{率} = \frac{\text{某时期内发生某事件的数量}}{\text{同期可能发生该事件的总数}} \times K$$

$K$ 为比例基数，通常取 $100\%$、$1000\%$、$10000/\text{万}$ 或 $100000/10\text{万}$。

**常见医学率**：
- 发病率、患病率、死亡率（详见[常用统计指标](常用统计指标.md)）
- 治愈率、有效率、病死率

### 2. 构成比（Proportion）

**构成比**反映某一事物内部各组成部分所占的比重或分布，各部分之和为 $100\%$。

$$\text{构成比} = \frac{\text{某组成部分的数量}}{\text{各组成部分的总数}} \times 100\%$$

**特点**：
- 各部分构成比之和为 $100\%$（或 $1$）
- 某一组成部分的变化会影响其他部分的构成比

### 3. 相对比（Ratio）

**相对比**是两个相关指标之比，反映两者的对比水平。

$$\text{相对比} = \frac{\text{甲指标}}{\text{乙指标}}$$

常用相对比：RR（相对危险度）、OR（比值比），详见[常用统计指标](常用统计指标.md)。

## Python 计算示例

```python
import pandas as pd
import numpy as np

# 临床疗效数据
data = pd.DataFrame({
    '组别': ['对照组', '治疗组'],
    '总例数': [120, 120],
    '治愈': [45, 72],
    '显效': [35, 28],
    '有效': [20, 12],
    '无效': [20, 8]
})

# 计算率和构成比
data['治愈率(%)'] = (data['治愈'] / data['总例数'] * 100).round(1)
data['有效率(%)'] = ((data['治愈'] + data['显效'] + data['有效'])
                     / data['总例数'] * 100).round(1)

# 治疗组内部构成比
治疗组 = data[data['组别'] == '治疗组'].iloc[:, 2:6].values[0]
data.loc[data['组别'] == '治疗组', '治愈构成比(%)'] = (
    治疗组 / 治疗组.sum() * 100).round(1)
data.loc[data['组别'] == '对照组', '治愈构成比(%)'] = (
    data[data['组别'] == '对照组'].iloc[:, 2:6].values[0]
    / data[data['组别'] == '对照组'].iloc[:, 2:6].values[0].sum() * 100
).round(1)

print("临床疗效分析")
print("=" * 70)
print(data[['组别', '总例数', '治愈率(%)', '有效率(%)']].to_string(index=False))

# 相对比：治疗组 vs 对照组
治愈率_治疗 = data.loc[data['组别'] == '治疗组', '治愈率(%)'].values[0]
治愈率_对照 = data.loc[data['组别'] == '对照组', '治愈率(%)'].values[0]
print(f"\n治愈率相对比（治疗/对照）= {治愈率_治疗 / 治愈率_对照:.2f}")
```

## 率的注意事项

### 1. 分母足够大

当分母过小时，率的波动较大，不宜直接比较。例如 $2/3$（$66.7\%$）和 $3/100$（$3\%$）的分母差异过大，前者的可靠性远低于后者。

### 2. 可比性

比较不同组别的率时，需注意：
- **内部构成是否一致**：如年龄、性别构成不同对率的比较有影响，需用**标准化法**校正（详见[标准化法](标准化法.md)）
- **诊断标准是否统一**
- **观察时间是否一致**

### 3. 率与构成比的区分

常见错误是将构成比当作率使用。例如：某医院死亡患者中，心脏病占比最高（构成比高），但并不能说明心脏病的死亡率高。

```python
# 构成比 ≠ 率的典型案例
hospital_data = pd.DataFrame({
    '疾病类型': ['冠心病', '肺癌', '肝癌', '糖尿病', '其他'],
    '死亡人数': [180, 120, 100, 50, 50],
    '患病人数': [5000, 800, 600, 4000, 10000]
})

hospital_data['构成比(%)'] = (hospital_data['死亡人数'] /
                                hospital_data['死亡人数'].sum() * 100).round(1)
hospital_data['死亡率(%)'] = (hospital_data['死亡人数'] /
                               hospital_data['患病人数'] * 100).round(2)

print("构成比与死亡率的区别")
print("=" * 65)
print(hospital_data.to_string(index=False))
print("\n→ 冠心病死亡构成比最高，但死亡率很低")
print("→ 肺癌死亡构成比低于冠心病，但死亡率远高于冠心病")
print("→ 说明：构成比不反映风险大小，仅反映构成分布")
```

### 4. 率的标准化

当比较两组总率时，若两组内部构成（如年龄、性别）不同，需进行标准化。详见[标准化法](标准化法.md)。

### 5. 率的标准误与可信区间

率的标准误：$SE_p = \sqrt{\frac{p(1-p)}{n}}$

率的 $95\%$ 可信区间：$p \pm 1.96 \times SE_p$

```python
from scipy.stats import norm

# 率的可信区间计算
n = 200
events = 60
p = events / n
se = np.sqrt(p * (1 - p) / n)

ci_lower = p - 1.96 * se
ci_upper = p + 1.96 * se

print(f"率 p = {p:.3f} ({events}/{n})")
print(f"标准误 SE = {se:.4f}")
print(f"95% 可信区间: ({ci_lower:.3f}, {ci_upper:.3f})")
```
