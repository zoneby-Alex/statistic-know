# 统计基础

概率论、抽样分布、数据描述构成生物医学统计的三大基石。

## 模块结构

| 子模块 | 内容 | 来源 |
|--------|------|------|
| 概率论 | 概率分布、抽样分布、正态分布 | 生物统计学第3章 / 医学统计学第3章 |
| 定量数据描述 | 频数分布、集中趋势、离散程度、统计图表 | 医学统计学第2、5章 |
| 定性数据描述 | 率与构成比、标准化法、常用指标 | 医学统计学第4章 / 生物统计学第2章 |

## 关键概念

```
统计描述 → 将原始数据转化为可理解的信息
概率论   → 描述随机性的数学语言
抽样分布 → 连接描述与推断的桥梁
```
