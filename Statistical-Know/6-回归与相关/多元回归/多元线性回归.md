# 多元线性回归（Multiple Linear Regression）

多元线性回归将直线回归推广到多个自变量的情形，用于研究多个因素对结局变量的联合影响。

## 模型形式

$$
y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_p x_p + \varepsilon
$$

- $y$：因变量（连续型）
- $x_1, x_2, \ldots, x_p$：自变量（可为连续、二分类或多分类哑变量）
- $\beta_0$：截距
- $\beta_j$：偏回归系数（partial regression coefficient）
- $\varepsilon$：随机误差，$\varepsilon \sim N(0, \sigma^2)$

矩阵形式：

$$
\mathbf{y} = \mathbf{X}\boldsymbol{\beta} + \boldsymbol{\varepsilon}
$$

OLS 解：

$$
\hat{\boldsymbol{\beta}} = (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\mathbf{y}
$$

## 偏回归系数的解释

**偏回归系数 $\beta_j$** 的含义：在其他自变量固定不变的条件下，$x_j$ 每变化一个单位，$y$ 的平均变化量。

这是回归分析的核心概念——$\beta_j$ 反映了 $x_j$ 对 $y$ 的**独立效应**（而非简单相关）。

## Python 实现

```python
import statsmodels.api as sm
import pandas as pd
import numpy as np

# 模拟数据：血压(y) ~ 年龄(x1) + BMI(x2) + 吸烟(x3)
np.random.seed(42)
n = 100
age = np.random.normal(50, 10, n)
bmi = np.random.normal(25, 4, n)
smoking = np.random.binomial(1, 0.3, n)  # 0=不吸烟, 1=吸烟

y = 80 + 0.5*age + 1.2*bmi + 8*smoking + np.random.normal(0, 5, n)

# 构建 DataFrame
data = pd.DataFrame({
    'SBP': y,
    'AGE': age,
    'BMI': bmi,
    'SMOKE': smoking
})

# 拟合模型
X = data[['AGE', 'BMI', 'SMOKE']]
X = sm.add_constant(X)  # 添加截距
model = sm.OLS(data['SBP'], X).fit()
print(model.summary())
```

输出关键部分解读：

| 变量 | Coef | Std.Err. | t | P>|t| | [0.025 | 0.975] |
|------|------|----------|---|-------|--------|-------|
| const | 80.23 | 2.15 | 37.3 | 0.000 | 75.97 | 84.50 |
| AGE | 0.48 | 0.04 | 12.0 | 0.000 | 0.40 | 0.56 |
| BMI | 1.18 | 0.10 | 11.8 | 0.000 | 0.98 | 1.38 |
| SMOKE | 7.92 | 0.89 | 8.9 | 0.000 | 6.15 | 9.68 |

## 变量选择方法

当自变量较多时，需要筛选重要变量。常用方法：

### 逐步回归（Stepwise）

```python
# 使用前向选择 + 后向剔除
def stepwise_selection(data, target, candidates, threshold_in=0.05, threshold_out=0.10):
    selected = []
    while True:
        # 前向选择
        remaining = [v for v in candidates if v not in selected]
        best_pval = float('inf')
        best_var = None
        for var in remaining:
            X = sm.add_constant(data[selected + [var]])
            model = sm.OLS(data[target], X).fit()
            pval = model.pvalues[var]
            if pval < best_pval:
                best_pval = pval
                best_var = var
        if best_pval <= threshold_in and best_var is not None:
            selected.append(best_var)
        else:
            break

        # 后向剔除
        changed = True
        while changed:
            changed = False
            X = sm.add_constant(data[selected])
            model = sm.OLS(data[target], X).fit()
            for var in selected:
                if model.pvalues[var] > threshold_out:
                    selected.remove(var)
                    changed = True
                    break
    return selected

selected_vars = stepwise_selection(
    data, 'SBP', ['AGE', 'BMI', 'SMOKE'],
    threshold_in=0.05, threshold_out=0.10
)
print(f"逐步回归选定变量: {selected_vars}")
```

### 其他筛选方法

| 方法 | 说明 | 优点 | 缺点 |
|------|------|------|------|
| 向前选择 | 从空模型开始，逐个加入显著变量 | 计算快 | 可能遗漏联合显著变量 |
| 向后剔除 | 从全模型开始，逐个剔除不显著变量 | 考虑全变量 | 变量多时不稳定 |
| 逐步回归 | 结合前两种 | 折中方案 | 多重比较问题 |
| LASSO | 带 L1 惩罚的回归 | 可同时进行变量选择+正则化 | 需调参 |

## 多重共线性与 VIF

多重共线性指自变量之间高度相关，导致回归系数估计不稳定、标准误增大。

### VIF（方差膨胀因子）

$$
VIF_j = \frac{1}{1 - R_j^2}
$$

其中 $R_j^2$ 为 $x_j$ 对其他自变量的回归决定系数。

经验标准：$VIF_j \geq 10$ 提示存在严重共线性；$VIF_j \geq 5$ 需注意。

```python
from statsmodels.stats.outliers_influence import variance_inflation_factor

X = data[['AGE', 'BMI', 'SMOKE']]
X = sm.add_constant(X)

vif_data = pd.DataFrame()
vif_data["Variable"] = X.columns
vif_data["VIF"] = [variance_inflation_factor(X.values, i)
                   for i in range(X.shape[1])]
print(vif_data)
```

### 共线性处理方法

- 删除高度相关的变量之一
- 使用主成分回归或岭回归
- 增加样本量
- 使用 LASSO 进行正则化

## 模型诊断补充

除直线回归的诊断外，多元回归还需检查：

1. **多重共线性**（VIF 检查）
2. **强影响点**（Cook's Distance、DFFITS）
3. **偏回归图**（检查各变量的线性关系）

```python
import matplotlib.pyplot as plt

# 偏回归图
fig = plt.figure(figsize=(12, 4))
sm.graphics.plot_partregress_grid(model, fig=fig)
plt.show()
```
