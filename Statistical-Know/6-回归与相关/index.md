# 回归与相关

研究变量间关系的统计方法。

## 内容

| 方法 | 结局变量 | 来源 |
|------|---------|------|
| 直线回归与相关 | 连续 | 医统第11章 / 生统第8章 |
| 多元线性回归 | 连续 | 医统第12章 / 生统第12章 |
| Logistic回归 | 二分类 | 医统第13章 |
| 通径分析 | 连续（因果路径） | 生统第13章 |

## 核心联系

```
直线回归：y = a + bx           （一个自变量）
多元回归：y = a + b₁x₁ + ...   （多个自变量）
Logistic：ln(p/1-p) = ...     （分类结局）
通径分析：分解直接/间接效应    （因果路径）
```
