# 直线回归（Simple Linear Regression）

回归分析研究一个因变量与一个或多个自变量之间的**定量依存关系**。直线回归（简单线性回归）是最基础的回归模型。

## 回归方程

直线回归模型假设 $Y$ 与 $X$ 之间存在线性关系：

$$
\hat{y} = a + bx
$$

- $\hat{y}$：回归预测值
- $a$：截距（$x=0$ 时 $y$ 的估计值）
- $b$：回归系数（斜率，$x$ 每变化 1 单位，$y$ 的平均变化量）

更完整的形式包含随机误差项：

$$
y = a + bx + \varepsilon, \quad \varepsilon \sim N(0, \sigma^2)
$$

## 最小二乘法（OLS）

最小二乘法通过最小化**残差平方和**来估计回归系数：

$$
\min \sum_{i=1}^{n} (y_i - \hat{y}_i)^2 = \min \sum_{i=1}^{n} (y_i - a - b x_i)^2
$$

求解得：

$$
b = \frac{\sum(x_i - \bar{x})(y_i - \bar{y})}{\sum(x_i - \bar{x})^2}, \quad a = \bar{y} - b\bar{x}
$$

## Python 实现：statsmodels

```python
import statsmodels.api as sm
import numpy as np
import pandas as pd

# 样本数据：体重(kg) vs 收缩压(mmHg)
weight = np.array([52, 58, 62, 65, 68, 70, 72, 75, 78, 80])
sbp    = np.array([110, 115, 120, 122, 128, 130, 135, 138, 142, 145])

# 添加截距项
X = sm.add_constant(weight)
model = sm.OLS(sbp, X).fit()

print(model.summary())
```

输出中包含：

- **coef**：回归系数 $a$（const）和 $b$（weight）
- **P>|t|**：回归系数的显著性检验 p 值
- **R-squared**：决定系数
- **F-statistic**：模型整体显著性

## 回归系数的 t 检验

对回归系数的显著性检验 $H_0: \beta = 0$：

$$
t = \frac{b}{SE(b)} \sim t_{n-2}
$$

```python
# 提取关键结果
print(f"截距 a = {model.params[0]:.4f}, p = {model.pvalues[0]:.4f}")
print(f"斜率 b = {model.params[1]:.4f}, p = {model.pvalues[1]:.4f}")
print(f"R²    = {model.rsquared:.4f}")
print(f"F     = {model.fvalue:.4f}, p = {model.f_pvalue:.4f}")
```

## 决定系数 R²

$$
R^2 = \frac{SS_{reg}}{SS_{total}} = 1 - \frac{SS_{res}}{SS_{total}}
$$

- $R^2$ 的取值范围为 $[0, 1]$
- 表示自变量 $X$ 能解释因变量 $Y$ 变异的比例
- 直线回归中 $R^2 = r^2$（$r$ 为 Pearson 相关系数）

## 回归诊断简介

OLS 回归有四个关键假设，需通过诊断图检查：

```python
import matplotlib.pyplot as plt

fig, axes = plt.subplots(2, 2, figsize=(10, 8))
residuals = model.resid
fitted = model.fittedvalues

# 1. 残差 vs 拟合值（检查线性 & 同方差性）
axes[0, 0].scatter(fitted, residuals)
axes[0, 0].axhline(y=0, color='r', linestyle='--')
axes[0, 0].set_xlabel("拟合值"); axes[0, 0].set_ylabel("残差")
axes[0, 0].set_title("残差 vs 拟合值")

# 2. Q-Q 图（检查正态性）
sm.qqplot(residuals, line='s', ax=axes[0, 1])
axes[0, 1].set_title("Q-Q 图")

# 3. Scale-Location 图
std_resid = model.get_influence().resid_studentized_internal
axes[1, 0].scatter(fitted, np.sqrt(np.abs(std_resid)))
axes[1, 0].set_xlabel("拟合值")
axes[1, 0].set_ylabel("√|标准化残差|")
axes[1, 0].set_title("Scale-Location 图")

# 4. Cook's Distance（检查强影响点）
sm.graphics.influence_plot(model, ax=axes[1, 1])
axes[1, 1].set_title("Cook's Distance")

plt.tight_layout()
plt.show()
```

### 四个关键诊断

| 诊断项 | 检验内容 | 方法 |
|-------|---------|------|
| 线性 | $Y$ 与 $X$ 是否线性相关 | 残差 vs 拟合值图 |
| 独立性 | 观测值是否独立 | Durbin-Watson 检验 |
| 同方差 | 残差方差是否恒定 | Breusch-Pagan 检验 |
| 正态性 | 残差是否正态分布 | Q-Q 图、Shapiro-Wilk 检验 |
