# 直线相关（Correlation Analysis）

相关分析研究两个变量之间**线性关系的方向和亲疏程度**，通过相关系数进行量化。

## Pearson 相关系数

### 定义与公式

Pearson 相关系数 $r$ 衡量两个连续变量之间的线性相关程度：

$$
r = \frac{\sum(x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum(x_i - \bar{x})^2 \sum(y_i - \bar{y})^2}} = \frac{Cov(x, y)}{s_x s_y}
$$

### 性质

- 取值范围：$[-1, 1]$
- $r = 1$：完全正相关；$r = -1$：完全负相关；$r = 0$：无线性相关
- 仅衡量**线性关系**，不能检测曲线关系
- 受**异常值**严重影响

### 强度判断（经验标准）

| |r| | 相关强度 |
|-----|---------|
| 0.00 – 0.20 | 极弱或无相关 |
| 0.20 – 0.40 | 弱相关 |
| 0.40 – 0.60 | 中等相关 |
| 0.60 – 0.80 | 强相关 |
| 0.80 – 1.00 | 极强相关 |

### t 检验

Pearson 相关系数的显著性检验 $H_0: \rho = 0$：

$$
t = r \sqrt{\frac{n-2}{1-r^2}} \sim t_{n-2}
$$

### Python 实现

```python
from scipy.stats import pearsonr
import numpy as np

# 年龄 vs 胆固醇水平
age = np.array([35, 42, 48, 51, 55, 60, 63, 67, 70, 74])
cholesterol = np.array([180, 195, 210, 220, 230, 240, 245, 260, 270, 280])

r, p_value = pearsonr(age, cholesterol)

print(f"Pearson r  = {r:.4f}")
print(f"p 值       = {p_value:.6f}")

# 手动计算 t 统计量
n = len(age)
t_stat = r * np.sqrt((n - 2) / (1 - r**2))
print(f"t 统计量   = {t_stat:.4f}")
```

## Spearman 等级相关

### 适用场景

- 数据不服从双变量正态分布
- 变量为等级资料（有序分类）
- 存在异常值
- 需要检测单调（而非严格线性）关系

### 原理

Spearman 相关系数 $r_s$ 计算的是**秩次**之间的 Pearson 相关系数：

$$
r_s = 1 - \frac{6\sum d_i^2}{n(n^2-1)}
$$

其中 $d_i$ 为每对观测值的秩次之差。

```python
from scipy.stats import spearmanr

# 等级数据：两评委对 10 个作品打分排序
judge_a = [1, 3, 2, 5, 4, 7, 6, 8, 10, 9]   # 排名
judge_b = [2, 1, 3, 4, 6, 5, 7, 9, 8, 10]

r_s, p_value = spearmanr(judge_a, judge_b)

print(f"Spearman r_s = {r_s:.4f}")
print(f"p 值         = {p_value:.4f}")
```

### Pearson vs Spearman 对比

| 特征 | Pearson | Spearman |
|------|---------|----------|
| **数据类型** | 连续变量 | 连续/等级 |
| **分布假设** | 双变量正态 | 无 |
| **检测关系** | 线性 | 单调 |
| **异常值影响** | 敏感 | 稳健 |
| **统计功效** | 高（线性关系） | 中（一般情况） |

## 相关与回归的关系

| 维度 | 相关分析 | 回归分析 |
|------|---------|---------|
| **变量地位** | 对等（无自变量/因变量区分） | 不对等（自变量 $X$，因变量 $Y$） |
| **核心指标** | 相关系数 $r$ | 回归系数 $b$ |
| **量的含义** | 相关方向与强度 | $X$ 变化 1 单位时 $Y$ 的变化量 |
| **预测功能** | 无 | 可进行预测 |
| **数学关系** | $r = b \cdot \frac{s_x}{s_y}$ | $b = r \cdot \frac{s_y}{s_x}$ |
| **$R^2$ 联系** | — | 直线回归中 $R^2 = r^2$ |

### 重要提醒

1. **相关 ≠ 因果**：即使 $r$ 显著，也不代表 $X$ 导致 $Y$，可能存在混杂因素
2. **需要检查散点图**：同一相关系数可能对应完全不同的数据形态（Anscombe's quartet）
3. **异常值影响**：单点异常值可能显著改变相关系数

```python
import matplotlib.pyplot as plt

# 检查散点图——永远第一步
plt.scatter(age, cholesterol)
plt.xlabel("年龄 (岁)"); plt.ylabel("胆固醇 (mg/dL)")
plt.title(f"Pearson r = {r:.3f}")
plt.show()
```
